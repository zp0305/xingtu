复合场问题解题套路 ​

模型编号: 23
难度: ⭐⭐⭐⭐⭐
核心: 电场 + 磁场 + 重力场 分段分析与衔接

模型识别 ​

关键词: 电场+磁场、E+B+g、粒子在复合场中运动

常见情境:

  • 粒子先在电场中加速/偏转，再进入磁场
  • 粒子同时在电场和磁场中运动
  • 考虑或不考虑重力

标准解题步骤 ​

步骤1：分析各场的作用 ​

【三场的作用】

重力场(g):
    - 始终存在，方向竖直向下
    - 是否考虑：题目说明或看粒子质量
    
电场(E):
    - 电场力 F电 = qE，方向与场强方向有关（看电荷正负）
    - 做功与路径无关，W = qU
    
磁场(B):
    - 洛伦兹力 F洛 = qvB，方向用左手定则
    - 始终不做功，只改变速度方向

步骤2：判断运动类型（分情况） ​

情况A：分区域运动（最常见）

粒子先在电场区域运动，再进入磁场区域

分析策略:
1. 分段分析：电场段 + 磁场段
2. 找衔接点：交界面处的速度（大小+方向）是关键！

情况B：同区域复合场（E+B+g同时存在）

分析策略:
1. 受力分析：重力 + 电场力 + 洛伦兹力
2. 判断合力情况:
   
   - 合力为零: 匀速直线（速度选择器原理）
     qvB = qE + mg（或减，看方向）
   
   - 合力不为零: 复杂曲线运动
     通常需要配速法或其他技巧（竞赛难度）

步骤3：电场段分析（类平抛或直线） ​

加速电场:

qU = ½mv² - ½mv₀²  （动能定理）

出射速度: v = √(v₀² + 2qU/m)

偏转电场（类平抛）:

水平: x = v₀t，匀速
竖直: y = ½at²，a = qE/m = qU/md

出射时:
水平速度: vx = v₀
竖直速度: vy = at = qUL/mdv₀
合速度大小: v = √(vx² + vy²)
速度方向: tanθ = vy/vx

关键: 出射时的速度方向！这是进入磁场的初始条件

步骤4：磁场段分析（圆周运动） ​

进入磁场后，只受洛伦兹力: F洛 = qvB

圆周运动:
半径: r = mv/qB  （v是进入磁场时的速度）
周期: T = 2πm/qB  （与速度无关）

【几何分析关键点】
1. 圆心位置：速度垂线交点
2. 圆心角：与偏转角的关系
3. 弦长：与半径的关系
4. 运动时间: t = θm/qB  （θ是圆心角，弧度）

步骤5：几何关系（难点！） ​

常见几何模型:

1. 直线边界进出:
   入射角 = 出射角（对称）
   偏转角 = 2×圆心角

2. 圆形磁场区域:
   指向圆心进，背离圆心出
   或：入射点、出射点、磁场圆心、轨迹圆心四边形

3. 临界问题:
   刚好射出、刚好不射出
   找与边界相切的临界轨迹

常用三角函数关系:

圆心角θ与偏转角的关系
弦长 L = 2r·sin(θ/2)
弧长 s = rθ

套路总结图 ​

复合场问题
    │
    ├──→ 分区域？还是同区域？
    │       │
    │       ├──→ 分区域：分段分析
    │       │       电场段 → 衔接点 → 磁场段
    │       │
    │       └──→ 同区域：受力分析
    │               合力为零？匀速
    │               合力不为零？复杂（配速法）
    │
    ├──→ 电场段
    │       类平抛：v₀水平，a = qE/m竖直
    │       或：直线加速
    │       【关键】出射速度大小+方向
    │
    ├──→ 磁场段
    │       圆周：r = mv/qB，T = 2πm/qB
    │       【关键】几何关系找圆心、圆心角
    │
    └──→ 衔接点（最重要！）
            电场出射速度 = 磁场入射速度
            大小、方向都要传递正确

常见组合套路 ​

套路1：电加速 + 磁偏转（质谱仪原理） ​

步骤:
1. 电场加速: qU = ½mv²  →  v = √(2qU/m)
2. 磁场偏转: r = mv/qB = √(2mU/q)/B
3. 打在屏上位置: x = 2r（直径）

结论: 同位素分离，质量不同半径不同

套路2：电偏转 + 磁偏转（速度选择器+分离） ​

步骤:
1. 速度选择器: qE = qvB₁ → v = E/B₁
   （只有速度为E/B₁的粒子能直线通过）
2. 进入磁场B₂: r = mv/qB₂

结论: 筛选特定速度的粒子，再按质荷比分离

套路3：磁偏转 + 电偏转（回旋加速器原理） ​

核心: 磁场中圆周，电场中加速
周期匹配: T电场 = T磁场 = 2πm/qB
最大动能: Ekm = q²B²R²/2m  （R是D形盒半径）

常用公式速查 ​

场景公式备注
电场加速qU = ½mv²动能定理
电场偏转y = qUL²/2mdv₀²类平抛
速度偏角tanθ = vy/vx = qUL/mdv₀²出射方向
磁场半径r = mv/qBv是入射速度
磁场周期T = 2πm/qB与v无关
运动时间t = θm/qBθ是圆心角（弧度）
弦长L = 2r·sin(θ/2)几何关系

常见陷阱 ​

陷阱正确做法
电场出射速度方向搞错类平抛的出射方向不是水平的，有偏角θ
磁场半径用错速度r = mv/qB中的v是进入磁场时的速度，不是初速度
几何关系找不到圆心速度垂线交点是圆心；入射点和出射点连线是弦
时间用t=s/v用t = θm/qB更简单，θ是圆心角
忘了重力看清题目：微观粒子一般不考虑重力，液滴/小球要考虑
正负电荷搞错负电荷受力方向与正电荷相反

快速检查清单 ​

  • [ ] 正确判断各力的方向（电场力看电荷正负，洛伦兹力用左手）
  • [ ] 电场段出射速度大小和方向计算正确
  • [ ] 衔接点速度传递正确（大小+方向）
  • [ ] 磁场中找圆心正确（速度垂线交点）
  • [ ] 几何关系正确（圆心角、弦长、半径关系）
  • [ ] 时间计算用t = θm/qB
  • [ ] 看清是否考虑重力

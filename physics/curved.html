曲线运动详解 ​

类别四：曲线运动与天体运动
学习时机：高一下学期 2-4月
核心能力：二维运动分析、向心力分析、万有引力应用

类别概述 ​

曲线运动是高中物理从直线到曲线的关键转折，核心思想是化曲为直——通过运动的合成与分解，将复杂的曲线运动转化为熟悉的直线运动处理。

三个模型的递进关系：

  • 模型13 圆周运动：曲线运动的基础，掌握向心力来源分析
  • 模型14 圆锥摆：圆周运动的典型应用，强调几何关系
  • 模型15 天体运动：万有引力提供向心力，是圆周运动的宇宙尺度推广

核心思想：

  • 曲线运动必有向心力（改变速度方向）
  • 向心力由实际力（重力、弹力、摩擦力、万有引力）提供
  • 天体运动中，万有引力=向心力=重力（近似）

模型13：圆周运动 ​

核心概念 ​

圆周运动的描述：

物理量公式关系
线速度v = 2πr/T = ωr描述运动快慢
角速度ω = 2π/T = v/r描述转动快慢
周期T = 2πr/v = 2π/ω转动一圈的时间
频率f = 1/T每秒转动的圈数

向心力公式（核心）：

F向 = mv²/r = mω²r = m(2π/T)²r = m(2πf)²r

向心加速度：

a向 = v²/r = ω²r

关键方法：向心力来源分析 ​

步骤：

  1. 确定圆周平面（水平/竖直/倾斜）
  2. 受力分析（重力、弹力、摩擦力）
  3. 指向圆心的合力 = 向心力

常见情景：

情景向心力来源注意点
水平转盘上的物块静摩擦力f_max = μN，有最大角速度限制
绳系小球水平面内转动绳拉力的水平分力与圆锥摆区分
汽车转弯静摩擦力（平地）或重力与支持力的合力（倾斜路面）有最大安全速度
过山车最高点重力+支持力（向下）临界：N=0，仅重力提供向心力

竖直平面内圆周运动的临界问题 ​

绳模型（只能拉）：

最高点临界：T = 0，仅重力提供向心力
mg = mv²/r → v临界 = √(gr)

通过条件：v ≥ √(gr)

杆模型（可拉可推）：

最高点临界：v = 0（杆可以支撑）
当 0 < v < √(gr)：杆向上推
当 v = √(gr)：杆无力
当 v > √(gr)：杆向下拉

对比记忆：

模型最高点最小速度特点
绳/环/外轨道√(gr)只能单向约束
杆/管/双轨道0可双向约束

典型例题 ​

【例题1】水平面内圆周运动

一根长为L的轻绳，一端固定，另一端系一质量为m的小球，在水平面内做匀速圆周运动，绳与竖直方向夹角为θ。

求：线速度v和周期T

解析：

受力分析：重力mg向下，绳拉力T沿绳向上

竖直方向平衡：Tcosθ = mg  →  T = mg/cosθ

水平方向（指向圆心）：Tsinθ = mv²/r
其中 r = Lsinθ（圆周半径）

代入：
(mg/cosθ)·sinθ = mv²/(Lsinθ)
g·tanθ = v²/(Lsinθ)

解得：
v = √(gLsinθ·tanθ) = √(gLsin²θ/cosθ)

周期：
T = 2πr/v = 2πLsinθ / √(gLsin²θ/cosθ) = 2π√(Lcosθ/g)

【例题2】竖直圆周运动临界

一质量为m的小球用长为L的轻绳系着，在竖直平面内做圆周运动。

(1) 求小球能通过最高点的最小速度 (2) 若小球在最高点速度为2√(gL)，求绳的拉力

解析：

(1) 最高点临界：仅重力提供向心力
    mg = mv²/L → v_min = √(gL)

(2) v = 2√(gL) > √(gL)，绳向下拉
    牛顿第二定律（向下为正）：
    mg + T = mv²/L = m·4gL/L = 4mg
    T = 3mg

FAQ小节 ​

Q1：向心力是实际存在的力吗？

不是。向心力是效果力，由实际力（重力、弹力、摩擦力）的合力或分力提供。分析时不能说"物体受向心力"，而要说"某某力提供向心力"。

Q2：为什么最高点临界速度是√(gr)？

当速度小于√(gr)时，需要的向心力mv²/r < mg，重力"过剩"，物体会掉下来。绳模型中绳子不能推，所以必须满足v ≥ √(gr)。

Q3：角速度和线速度什么时候用哪个公式？

  • 同轴转动（如转盘上各点）：角速度ω相同，用v = ωr比较线速度
  • 皮带/齿轮传动：线速度v相同，用ω = v/r比较角速度

模型14：圆锥摆 ​

核心模型 ​

结构：小球在水平面内做匀速圆周运动，绳与竖直方向成θ角，形成圆锥面。

受力：重力mg向下，绳拉力T沿绳向上

关键方程：

竖直平衡：Tcosθ = mg
水平向心力：Tsinθ = mω²r

其中 r = Lsinθ（圆周半径）

联立得：
tanθ = ω²r/g = ω²Lsinθ/g
ω² = g/(Lcosθ)

周期：
T = 2π/ω = 2π√(Lcosθ/g)

重要结论：

  • 周期只与L和θ有关，与质量无关
  • θ越大，cosθ越小，周期越小，角速度越大
  • 同一高度的圆锥摆周期相同（与绳长无关）

与单摆的区别 ​

对比圆锥摆单摆
运动水平面匀速圆周竖直平面变速摆动
周期公式T = 2π√(Lcosθ/g)T = 2π√(L/g)
速度大小恒定变化（最低点最大）
加速度向心加速度（水平）切向+向心加速度

模型15：天体运动 ​

核心模型 ​

基本假设：天体运动近似为匀速圆周运动，万有引力提供向心力

万有引力定律：

F = GMm/r²

G = 6.67×10⁻¹¹ N·m²/kg²（引力常量）

核心方程：万有引力 = 向心力

GMm/r² = mv²/r = mω²r = m(2π/T)²r

常用结论（需熟记） ​

线速度：

v = √(GM/r)

轨道半径越大，线速度越小（高轨低速）

角速度：

ω = √(GM/r³)

轨道半径越大，角速度越小（高轨慢角速）

周期：

T = 2π√(r³/GM)

轨道半径越大，周期越大（高轨大周期）——开普勒第三定律

向心加速度：

a = GM/r²

轨道半径越大，加速度越小

口诀：高轨低速大周期

黄金代换 ​

适用条件：星球表面或近地轨道（忽略自转）

万有引力 ≈ 重力
GMm/R² = mg

GM = gR²（黄金代换）

用于：不知道M，但知道g和R时

典型情景 ​

情景1：比较不同卫星参数

已知：卫星A轨道半径r₁，卫星B轨道半径r₂，且r₁ < r₂

比较：v₁ vs v₂，ω₁ vs ω₂，T₁ vs T₂，a₁ vs a₂

解析：

由 v = √(GM/r)：r小则v大 → v₁ > v₂
由 ω = √(GM/r³)：r小则ω大 → ω₁ > ω₂
由 T = 2π√(r³/GM)：r小则T小 → T₁ < T₂
由 a = GM/r²：r小则a大 → a₁ > a₂

情景2：求星球质量或密度

已知：卫星绕星球做匀速圆周运动，轨道半径r，周期T

求：星球质量M

解析：

万有引力 = 向心力
GMm/r² = m(2π/T)²r

解得：
M = 4π²r³/GT²

若已知星球半径R，求密度：
ρ = M/V = M/(4/3·πR³) = 3πr³/GT²R³

近地卫星(r≈R)：
ρ = 3π/GT²（只需测周期即可求密度）

情景3：第一宇宙速度

定义：卫星在星球表面附近绕星球做匀速圆周运动的速度

万有引力 = 向心力
GMm/R² = mv²/R

v₁ = √(GM/R) = √(gR)

地球：v₁ ≈ 7.9 km/s

三种宇宙速度（地球）：

宇宙速度数值意义
第一7.9 km/s最小发射速度，最大环绕速度
第二11.2 km/s脱离地球引力束缚
第三16.7 km/s脱离太阳系

卫星变轨问题 ​

基本规律：

  • 点火加速 → 做离心运动 → 轨道升高 → 机械能增加
  • 点火减速 → 做向心运动 → 轨道降低 → 机械能减少

高轨 vs 低轨对比（同一卫星）：

参数高轨低轨变化
线速度v小大减速升轨
角速度ω小大减速升轨
周期T大小升轨周期变大
机械能E大小升轨需点火加速
动能Ek小大速度减小
势能Ep大小高度增加

注意：升轨后虽然速度减小，但动能减小、势能增加更多，总机械能增加（发动机做功）。

典型例题 ​

【例题】同步卫星

地球同步卫星距地面高度h，地球半径R，地球表面重力加速度g。

求：同步卫星的线速度v和周期T

解析：

同步卫星特点：周期T = 24h = 86400s（与地球自转周期相同）
轨道半径：r = R + h

由万有引力 = 向心力：
GMm/r² = m(2π/T)²r

解得：
r³ = GMT²/4π²

由黄金代换 GM = gR²：
r³ = gR²T²/4π²

线速度：
v = 2πr/T = 2π(R+h)/T

FAQ小节 ​

Q1：卫星的向心加速度与重力加速度有什么关系？

在轨道处：a向 = GM/r²（万有引力产生的加速度） 在地面：g = GM/R² 所以 a向/g = R²/r²，轨道越高，向心加速度越小

Q2：为什么同步卫星只能定点在赤道上空？

同步卫星要与地球自转同步，其轨道圆心必须在地球自转轴上。只有赤道轨道满足这一条件，倾斜轨道或极地轨道的圆心不在地轴上，无法与地球同步。

Q3：卫星变轨时，为什么加速反而轨道升高、速度减小？

点火加速瞬间，卫星速度增大，需要的向心力mv²/r增大，但万有引力不变，不足以提供向心力，卫星做离心运动，轨道升高。 到达新高轨后，由v = √(GM/r)，r增大导致v减小，最终速度比低轨还小。

类别总结 ​

曲线运动三类问题对比：

问题类型关键分析核心公式
圆周运动向心力来源F向 = mv²/r = mω²r
圆锥摆几何关系T = 2π√(Lcosθ/g)
天体运动万有引力=向心力v = √(GM/r)，T = 2π√(r³/GM)

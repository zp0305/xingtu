能量动量详解 ​

类别五：能量与动量
学习时机：高一下学期 4-7月
核心能力：功和能的关系、动量守恒应用、综合问题分析

类别概述 ​

能量和动量是高中物理的两大核心工具，它们提供了分析力学问题的两种视角：

  • 能量视角：关注过程的始末，不关注中间细节（标量，只有大小）
  • 动量视角：关注相互作用前后的状态，适用于碰撞、爆炸（矢量，有方向）

五个模型的递进关系：

  • 模型16 动能定理：处理单个物体的能量变化
  • 模型17 机械能守恒：处理只有重力/弹力做功的系统
  • 模型18 功能关系：处理有摩擦力等非保守力的情况
  • 模型19 动量守恒：处理碰撞、爆炸等相互作用
  • 模型20 能量动量综合：处理复杂多过程问题

核心思想：

  • 能量守恒（广义）：能量不会消失，只会转化
  • 动量守恒：系统不受外力或外力合力为零时，总动量守恒

模型16：动能定理 ​

核心概念 ​

功的定义：

W = F·s·cosθ

θ = 0°：力与位移同向，做正功
θ = 90°：力与位移垂直，不做功
θ = 180°：力与位移反向，做负功

动能：

Ek = ½mv²

动能定理：

W合 = ΔEk = Ek末 - Ek初 = ½mv₂² - ½mv₁²

关键理解：

  • 动能定理是标量方程，无方向
  • 适用于直线运动和曲线运动
  • 适用于恒力做功和变力做功（变力用功的定义积分）
  • 只看初末状态，不看中间过程（优势！）

典型应用 ​

应用1：求变力做功

若力是变力或路径复杂，直接用W=Fs困难，可用动能定理间接求。

W合 = ΔEk
W其他已知力 + W待求 = ΔEk
→ 可求出待求力的功

应用2：曲线运动求速度

如平抛运动、圆周运动，用牛顿第二定律求加速度再积分复杂，用动能定理简单。

例题：小球从高度h处静止滑下光滑曲面，求最低点的速度。

只有重力做功：
W合 = mgh = ΔEk = ½mv² - 0

v = √(2gh)

常见力做功特点 ​

力做功公式特点
重力WG = mgh只看高度差，与路径无关
弹力W弹 = ½kx₁² - ½kx₂²弹性势能变化量的负值
摩擦力Wf = -f·s路程与路径有关，生热Q = f·s相对
万有引力W引 = GMm(1/r₂ - 1/r₁)与路径无关
电场力W电 = qU = qEd与路径无关

模型17：机械能守恒 ​

核心概念 ​

机械能：

E = Ek + Ep

Ek = ½mv²（动能）
Ep重力 = mgh（重力势能）
Ep弹性 = ½kx²（弹性势能）

机械能守恒条件：

只有重力或弹力做功，其他力不做功或做功代数和为零

ΔE = 0 → Ek₁ + Ep₁ = Ek₂ + Ep₂

判断方法：

  1. 看是否有摩擦力（有摩擦→不守恒，转化为内能）
  2. 看是否有外力做功（如人推、发动机牵引）
  3. 看是否有爆炸、碰撞（一般机械能不守恒）

典型应用 ​

应用：抛体运动、单摆、弹簧振子

例题：小球从A点静止释放，沿光滑圆弧轨道滑到B点，求B点速度。

光滑→无摩擦，只有重力做功→机械能守恒

设A点为势能零点：
EkA + EpA = EkB + EpB
0 + mgR = ½mv² + 0

v = √(2gR)

机械能守恒 vs 动能定理 ​

对比机械能守恒动能定理
研究对象系统（物体+地球+弹簧）单个物体
方程形式Ek₁+Ep₁ = Ek₂+Ep₂W合 = ΔEk
适用条件只有重力/弹力做功任何情况
优势不涉及力的细节可求功和能的定量关系

建议：能用机械能守恒时先用（简单），不能用时用动能定理（通用）。

模型18：功能关系 ​

核心概念 ​

功能关系本质是能量守恒的另一种表述：

功是能量转化的量度

常见功能关系：

做功的力能量转化功的计算
重力重力势能↔动能WG = -ΔEp重 = mgh
弹力弹性势能↔动能W弹 = -ΔEp弹
摩擦力机械能→内能Wf = -Q = -f·s相对
电场力电势能↔动能W电 = -ΔEp电
合力动能变化W合 = ΔEk
非保守力机械能变化W非保守 = ΔE机

核心公式：

W合 = ΔEk（动能定理）
W非保守力 = ΔE机（功能原理）
W摩擦力 = -Q（热量）

典型应用：板块模型热量计算 ​

问题：质量为m的物块以v₀滑上静止木板M，最终共速，求产生的热量。

解析：

能量守恒：系统动能损失 = 热量
Q = ½mv₀² - ½(m+M)v共²

或用摩擦力做功：
Q = f·s相对

两种方法等价

能量守恒方程写法 ​

通用写法：

E初 = E末 + E损失

或：
减少的能量 = 增加的能量

例题：物块从高h处滑下，经过粗糙水平面（摩擦系数μ，长度L），求最终停止位置。

能量守恒：
重力势能减少 = 摩擦力做功（生热）
mgh = μmgL

解得：L = h/μ

模型19：动量守恒 ​

核心概念 ​

动量：

p = mv（矢量，方向与速度相同）

冲量：

I = F·t = Δp（矢量）

动量守恒定律：

系统不受外力或外力合力为零时，总动量守恒

m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'

适用条件：

  1. 系统不受外力（理想情况）
  2. 外力合力为零（如水平方向动量守恒）
  3. 内力远大于外力（短暂过程，如碰撞、爆炸）

碰撞类型 ​

碰撞类型特点方程
弹性碰撞动能守恒动量守恒 + 动能守恒
非弹性碰撞动能不守恒只有动量守恒
完全非弹性碰撞共速，动能损失最大动量守恒 + v₁'=v₂'

弹性碰撞公式（记住结论）：

v₁' = [(m₁-m₂)v₁ + 2m₂v₂]/(m₁+m₂)
v₂' = [(m₂-m₁)v₂ + 2m₁v₁]/(m₁+m₂)

特殊情况：v₂=0（静止目标）
v₁' = (m₁-m₂)v₁/(m₁+m₂)
v₂' = 2m₁v₁/(m₁+m₂)

记忆口诀：

  • 等质量交换速度：m₁=m₂时，v₁'=0，v₂'=v₁
  • 大撞小，同向走；小撞大，反弹回

典型应用 ​

应用1：人船模型

问题：质量为m的人从质量为M、长为L的船的一端走到另一端，求船的位移。

解析：

水平方向动量守恒（系统不受外力）：
设人速度v（向右），船速度V（向左）

mv = MV（动量大小相等，方向相反）

两边乘时间t：
m·x人 = M·x船

且 x人 + x船 = L

解得：
x船 = mL/(m+M)（向左）
x人 = ML/(m+M)（向右）

结论：位移与质量成反比

应用2：反冲运动

问题：总质量为M的火箭（含燃料）静止，喷出质量为m的气体，气体速度为v（相对地面），求火箭速度。

解析：

动量守恒（初态总动量为0）：
0 = mv + (M-m)V

V = -mv/(M-m)（负号表示与气体反向）

模型20：能量动量综合 ​

核心思想 ​

复杂问题的分析框架：

1. 分段分析：按时间或空间分成若干段
2. 每段选择合适工具：
   - 涉及时间t和力F → 用动量定理（Ft = Δp）
   - 涉及位移s和力F → 用动能定理（Fs = ΔEk）
   - 碰撞、爆炸 → 用动量守恒
   - 只有重力做功 → 用机械能守恒
3. 找衔接点：前段末态 = 后段初态

典型综合问题 ​

问题类型1：碰撞+滑动

例题：物块A（质量m₁，速度v₀）与静止物块B（质量m₂）弹性碰撞后，B滑上静止木板C（质量m₃），求最终三者速度。

解析：

阶段1：A与B弹性碰撞
动量守恒：m₁v₀ = m₁v₁ + m₂v₂
动能守恒：½m₁v₀² = ½m₁v₁² + ½m₂v₂²
→ 解得v₁、v₂

阶段2：B在C上滑动
水平动量守恒：m₂v₂ = (m₂+m₃)v共
能量损失：Q = ½m₂v₂² - ½(m₂+m₃)v共² = f·s相对
→ 解得v共、滑行距离

问题类型2：弹簧+碰撞

例题：物块A以v₀撞向连有弹簧的物块B，求弹簧最大压缩量。

解析：

关键点：弹簧最大压缩时，A和B速度相同（相对速度为0）

动量守恒：m₁v₀ = (m₁+m₂)v共
→ v共 = m₁v₀/(m₁+m₂)

能量守恒：
½m₁v₀² = ½(m₁+m₂)v共² + Ep弹max
→ Ep弹max = ½m₁v₀² - ½(m₁+m₂)[m₁v₀/(m₁+m₂)]²
         = ½·m₁m₂v₀²/(m₁+m₂)

最大压缩量由Ep弹max = ½kx²求得

解题策略总结 ​

问题特征首选工具次选工具
碰撞过程动量守恒动能守恒（弹性）
涉及时间动量定理牛顿定律
涉及位移动能定理能量守恒
往复运动能量守恒动量定理（分次）
多物体系统动量守恒+能量守恒隔离法+动能定理

类别总结 ​

能量vs动量选择指南：

情境用能量用动量两者都用
求速度大小✅✅
求位移/高度✅❌
求时间❌✅
碰撞问题✅（弹性）✅✅（综合）
板块模型✅（热量）✅（共速）✅

高频考点：

  1. 动能定理求变力做功
  2. 机械能守恒条件判断
  3. 完全非弹性碰撞（共速）
  4. 人船模型（动量守恒+位移关系）
  5. 能量动量综合（多过程分段）

与后续知识联系：

  • 能量守恒 → 热力学第一定律
  • 动量守恒 → 原子物理（衰变、核反应）
  • 功能关系 → 电势能、电功

FAQ小节 ​

Q1：什么时候用动能定理，什么时候用机械能守恒？

  • 能用机械能守恒时优先用（简单，不涉及力的细节）
  • 有摩擦力或其他非保守力做功时，用动能定理
  • 动能定理更通用，机械能守恒是特殊情况

Q2：动量守恒和动能定理能同时用吗？

可以，而且经常需要同时用。动量守恒处理矢量关系（方向），动能定理处理标量关系（大小）。碰撞问题中，动量守恒是必须的，弹性碰撞还需动能守恒。

Q3：人船模型为什么位移与质量成反比？

因为水平动量守恒：mv = MV → m(vt) = M(Vt) → mx₁ = Mx₂，所以x₁/x₂ = M/m。质量大的位移小，质量小的位移大。

Q4：弹簧最大压缩量为什么对应共速？

弹簧压缩量最大时，两物体间没有相对运动（如果有相对运动，弹簧还在压缩）。相对速度为0意味着两物体速度相同。

学习建议：

  1. 先熟练掌握单个模型（动能定理、动量守恒）
  2. 再练习两个模型的结合（碰撞+滑动）
  3. 最后攻克三模型综合（弹簧+碰撞+滑动）
  4. 多画过程图，明确每个阶段用什么工具

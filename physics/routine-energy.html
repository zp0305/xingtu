能量动量综合套路 ​

模型编号: 20
难度: ⭐⭐⭐⭐⭐
核心: 多过程分析 + 能量动量工具选择

模型识别 ​

关键词: 多过程、碰撞、滑动、能量转化、动量守恒

常见情境:

  • 碰撞后滑上木板/曲面
  • 弹簧连接+碰撞
  • 爆炸后多物体运动
  • 多次碰撞问题

核心工具选择 ​

什么时候用能量？什么时候用动量？ ​

问题特征用能量用动量两者都用
涉及位移/高度✅❌
涉及时间❌✅
碰撞过程✅（弹性）✅✅
求速度大小✅✅
求共同速度✅✅✅
求热量/做功✅❌
多物体相互作用❌✅

核心方程 ​

能量守恒（系统）:

E初 = E末 + Q（热量）

或：减少的能量 = 增加的能量

动量守恒（系统）:

m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'（矢量式）

动能定理（单个物体）:

W合 = ΔEk

多过程分析步骤 ​

步骤1：划分过程 ​

按时间顺序或空间位置划分阶段：

例：物块A撞B，B滑上木板C

过程1：A与B碰撞
过程2：B在C上滑动至共速

步骤2：每个过程选工具 ​

过程类型推荐工具原因
碰撞（短暂）动量守恒内力大，忽略外力
滑动（有摩擦）能量守恒+动量守恒有热量产生
弹簧作用能量守恒+动量守恒弹力是内力
圆周运动能量守恒+牛顿定律涉及向心力

步骤3：找衔接点 ​

关键原则：

前段末态 = 后段初态

即：前段的末速度 = 后段的初速度
     前段的末位置 = 后段的初位置

步骤4：联立求解 ​

列方程策略：

  • 有几个未知量，列几个独立方程
  • 动量守恒（矢量）可分解为x、y两个方向
  • 能量守恒是标量，只有一个方程

典型问题类型 ​

类型1：碰撞+滑动（板块） ​

问题: A以v₀撞静止B，B滑上静止木板C，求最终速度及热量

解法:

过程1：A与B碰撞

动量守恒：m_A v₀ = m_A v_A + m_B v_B
若弹性：½m_A v₀² = ½m_A v_A² + ½m_B v_B²

解得v_A、v_B（v_B为B滑上C的初速度）

过程2：B在C上滑动

动量守恒：m_B v_B = (m_B + m_C)v共
能量守恒：½m_B v_B² = ½(m_B+m_C)v共² + Q
其中 Q = f·s相对 = μmg·s相对

可求v共和滑动距离s相对

类型2：弹簧+碰撞 ​

问题: A以v₀撞向连有弹簧的静止B，求弹簧最大压缩量

解法:

关键点：弹簧最大压缩时，A和B速度相同（相对速度为0）

动量守恒：m_A v₀ = (m_A + m_B)v共
→ v共 = m_A v₀/(m_A + m_B)

能量守恒：
½m_A v₀² = ½(m_A+m_B)v共² + Ep弹max

解得Ep弹max = ½m_A v₀² - ½(m_A+m_B)[m_A v₀/(m_A+m_B)]²
           = ½·m_A m_B v₀²/(m_A+m_B)

最大压缩：x_max = √(2Ep弹max/k)

类型3：爆炸/反冲+平抛 ​

问题: 炮弹在高h处水平爆炸成两块，求落地点距离

解法:

过程1：爆炸（瞬间）

水平动量守恒：
Mv₀ = m₁v₁ + m₂v₂

爆炸能量：化学能→动能（能量增加）

过程2：平抛（各自）

时间：t = √(2h/g)（相同）
水平位移：x₁ = v₁t，x₂ = v₂t
落点距离：Δx = |x₁ - x₂| = |v₁ - v₂|t

常见陷阱 ​

陷阱正确做法
碰撞过程用能量守恒（非弹性）非弹性碰撞机械能不守恒，只能用动量守恒
忘记热量计算有摩擦时，动能损失=热量Q=f·s相对
动量守恒方向搞错矢量式，设正方向，注意正负号
混淆系统和单体动量守恒对系统，动能定理对单体
弹簧最大压缩条件错共速时压缩最大，不是速度为零

快速检查清单 ​

  • [ ] 正确划分了过程阶段
  • [ ] 每个过程选对了工具（能量/动量）
  • [ ] 动量守恒方程设了正方向
  • [ ] 能量守恒包含了所有能量形式（含热量）
  • [ ] 衔接点速度/位置传递正确
  • [ ] 未知量个数=独立方程个数

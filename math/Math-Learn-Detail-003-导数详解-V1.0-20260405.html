二、概念定义 ​

2.1 平均变化率 ​

定义：函数f(x)从x₁到x₂的平均变化率为

Δy/Δx = [f(x₂)-f(x₁)]/(x₂-x₁)

几何意义：割线斜率

示例：

  • 位移从s(t₁)到s(t₂)的平均速度
  • 成本从C(q₁)到C(q₂)的平均变化

2.2 瞬时变化率与导数 ⭐ ​

导数定义：

f'(x₀) = lim(Δx→0) [f(x₀+Δx)-f(x₀)]/Δx

理解：

  • 平均变化率当Δx→0时的极限
  • 瞬时变化率
  • 几何意义：切线斜率

导函数：f'(x)或y'或dy/dx

  • 将每一点的导数作为函数值，形成的新函数

2.3 导数的几何意义 ​

核心：函数y=f(x)在点(x₀,f(x₀))处的导数f'(x₀)就是曲线在该点处切线的斜率。

切线方程：

y - f(x₀) = f'(x₀)(x - x₀)

法线方程（垂直于切线）：

y - f(x₀) = -[1/f'(x₀)](x - x₀)  （f'(x₀)≠0）

2.4 基本初等函数的导数 ​

函数导数备注
常数 c0
幂函数 xⁿnxⁿ⁻¹n为实数
指数 eˣeˣ特殊的：导数等于自身
指数 aˣaˣ·ln a
对数 ln x1/xx>0
对数 logₐx1/(x·ln a)
正弦 sin xcos x
余弦 cos x-sin x注意负号

记忆技巧：

  • sin→cos→(-sin)→(-cos)→sin（循环求导）
  • (eˣ)' = eˣ，因为e的定义就是使得这个成立的数

2.5 导数的四则运算 ​

加法：

[f(x) ± g(x)]' = f'(x) ± g'(x)

乘法：

[f(x)·g(x)]' = f'(x)g(x) + f(x)g'(x)

除法：

[f(x)/g(x)]' = [f'(x)g(x) - f(x)g'(x)] / [g(x)]²

记忆：

  • 乘法："前导后不导 + 前不导后导"
  • 除法："上导下不导 - 上不导下导，除以平方"

2.6 复合函数求导（链式法则）⭐ ​

法则：若y=f(u)，u=g(x)，则

dy/dx = dy/du · du/dx = f'(u) · g'(x)

口诀："由外到内，层层求导，连乘起来"

示例：

y = sin(2x+1)
设u = 2x+1，则y = sin u
y' = cos u · 2 = 2cos(2x+1)

常见复合函数：

  • e^(ax+b) → a·e^(ax+b)
  • ln(ax+b) → a/(ax+b)
  • sin(ax+b) → a·cos(ax+b)
  • (ax+b)ⁿ → n·a·(ax+b)ⁿ⁻¹

三、导数应用 ⭐ ​

3.1 函数的单调性 ​

定理：

  • f'(x) > 0 → f(x)单调递增
  • f'(x) < 0 → f(x)单调递减
  • f'(x) = 0 → 可能是极值点

求单调区间步骤：

  1. 求f'(x)
  2. 解f'(x)=0，得临界点
  3. 列表分析各区间f'(x)的符号
  4. 确定单调区间

3.2 函数的极值 ​

定义：

  • 极大值：在x₀附近，f(x₀)比两侧都大
  • 极小值：在x₀附近，f(x₀)比两侧都小

求极值步骤：

  1. 求f'(x)
  2. 解f'(x)=0，得驻点
  3. 判断驻点两侧导数符号变化
    • 左正右负→极大值
    • 左负右正→极小值
    • 同号→不是极值点

注意：

  • 极值是局部概念，不是最值
  • f'(x₀)=0只是必要条件，不是充分条件

3.3 函数的最值 ​

闭区间上求最值：

  1. 求f'(x)，找驻点
  2. 求驻点和端点的函数值
  3. 比较大小，最大的为最大值，最小的为最小值

实际应用中的最值问题：

  • 面积最大、体积最大
  • 成本最低、利润最大
  • 用料最省

建模步骤：

  1. 设变量，建立目标函数
  2. 确定定义域（实际约束）
  3. 求导，找极值点
  4. 比较极值点和端点，确定最值

3.4 函数的零点 ​

利用导数研究零点：

  1. 求f'(x)，确定单调性
  2. 找极值点，计算极值
  3. 结合极值符号和区间端点值，判断零点个数

零点存在性：

  • 若连续函数在区间端点异号，则区间内至少有一个零点
  • 结合单调性可确定唯一性

3.5 不等式证明 ​

常用方法：

法1：构造函数

  • 要证f(x) > g(x)，设h(x) = f(x) - g(x)
  • 证明h(x) > 0（求h(x)的最小值）

法2：利用单调性

  • 证明f(x)在[a,+∞)递增，且f(a)≥0
  • 则f(x)≥f(a)≥0

法3：放缩法

  • 利用已知不等式（如eˣ ≥ x+1，ln x ≤ x-1）

3.6 恒成立问题 ​

常见类型：

类型1：f(x) ≥ 0在区间I上恒成立

  • 求f(x)在I上的最小值≥0

类型2：f(x) ≤ 0在区间I上恒成立

  • 求f(x)在I上的最大值≤0

类型3：f(x) ≥ g(x)恒成立

  • 转化为h(x) = f(x) - g(x) ≥ 0

分离参数法：

  • 若不等式可化为a ≥ φ(x)或a ≤ φ(x)
  • 则a ≥ φ(x)ₘₐₓ 或 a ≤ φ(x)ₘᵢₙ

四、典型例题 ​

例题1：导数计算 ​

题目：求下列函数的导数

  • (1) y = x³ - 2x² + 3x - 1
  • (2) y = x·eˣ
  • (3) y = ln(2x+1)

解答：

(1) y' = 3x² - 4x + 3

(2) y' = (x)'·eˣ + x·(eˣ)' = eˣ + x·eˣ = (1+x)eˣ

(3) 设u = 2x+1，则y = ln u
    y' = (1/u) · 2 = 2/(2x+1)

例题2：切线方程 ​

题目：求曲线y = x³ - 2x在点(1,-1)处的切线方程。

解答：

y' = 3x² - 2
x=1时，y' = 3-2 = 1

切线方程：y - (-1) = 1·(x-1)
即 y = x - 2

例题3：单调性与极值 ​

题目：求f(x) = x³ - 3x²的单调区间和极值。

解答：

f'(x) = 3x² - 6x = 3x(x-2)

令f'(x)=0，得x=0或x=2

列表分析：
x       (-∞,0)   0   (0,2)   2   (2,+∞)
f'(x)      +      0     -     0      +
f(x)      ↗    极大   ↘   极小    ↗

单调增区间：(-∞,0)和(2,+∞)
单调减区间：(0,2)
极大值：f(0) = 0
极小值：f(2) = 8-12 = -4

例题4：最值应用 ​

题目：用长为60cm的铁丝围成一个矩形，问长宽各为多少时面积最大？

解答：

设长为x，则宽为(30-x)
面积S = x(30-x) = 30x - x²  (0<x<30)

S' = 30 - 2x
令S'=0，得x=15

S'' = -2 < 0，故x=15是极大值点

当x=15时，宽也是15（正方形）
最大面积S = 225 cm²

五、配套练习 ​

基础练习（必做） ​

练习1：求下列函数的导数

  • (1) y = 3x⁴ - 2x³ + x - 5
  • (2) y = x²·ln x
  • (3) y = e^(3x-1)

练习2：求曲线y = x² + 1在点(2,5)处的切线方程。

练习3：求f(x) = x³ - 6x² + 9x的单调区间和极值。

练习4：证明当x > 0时，eˣ > x+1。

参考答案：

  • 练习1：(1) 12x³-6x²+1；(2) 2x·ln x + x；(3) 3e^(3x-1)
  • 练习2：y = 4x - 3
  • 练习3：增区间(-∞,1)和(3,+∞)，减区间(1,3)；极大值4，极小值0
  • 练习4：设f(x)=eˣ-x-1，f'(x)=eˣ-1>0(x>0)，f(0)=0，∴f(x)>0

提高练习（选做） ​

练习5：求f(x) = x·e⁻ˣ的单调区间、极值和最值。

练习6：已知f(x) = x³ + ax² + bx + c在x=1处有极值-1，且f(0)=0，求a,b,c。

六、FAQ常见问题 ​

FAQ 1：复合函数求导总漏乘内层导数 ​

错误示例：

  • y = sin(2x)，y' = cos(2x) ✗（漏乘2）
  • 正确：y' = 2cos(2x)

检查方法：

  • 每次求导后检查：是否还有内层函数需要求导
  • 口诀："由外到内，层层求导，连乘起来"

FAQ 2：导数与单调性的关系总搞反 ​

记忆：

  • f'(x) > 0 → f(x) 增（导数为正，函数上升）
  • f'(x) < 0 → f(x) 减（导数为负，函数下降）

注意：

  • f'(x) > 0是充分条件，不是必要条件
  • f(x)递增时，可能有f'(x)=0的点（如f(x)=x³在x=0处）

FAQ 3：极值点和最值点分不清 ​

区别：

  • 极值点：局部概念，只要在附近最大/最小即可
  • 最值点：全局概念，整个区间内最大/最小

联系：

  • 开区间内，最值点一定是极值点
  • 闭区间内，最值点可能是极值点或端点

FAQ 4：恒成立问题不知道怎么转化 ​

转化策略：

原问题转化
f(x) ≥ 0恒成立f(x)ₘᵢₙ ≥ 0
f(x) ≤ 0恒成立f(x)ₘₐₓ ≤ 0
f(x) ≥ g(x)恒成立[f(x)-g(x)]ₘᵢₙ ≥ 0
存在x使f(x) ≥ 0f(x)ₘₐₓ ≥ 0

注意：

  • "恒成立"对应最值
  • "存在"对应极值

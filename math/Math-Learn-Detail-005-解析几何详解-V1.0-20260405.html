二、概念定义 ​

2.1 直线的方程 ⭐ ​

倾斜角：直线向上的方向与x轴正方向的夹角α，0°≤α<180°

斜率：k = tanα（α≠90°）

  • 两点A(x₁,y₁), B(x₂,y₂)的斜率：k = (y₂-y₁)/(x₂-x₁)

直线方程的几种形式：

形式方程适用条件
点斜式y-y₀ = k(x-x₀)已知点和斜率
斜截式y = kx+b已知斜率和y轴截距
两点式(y-y₁)/(y₂-y₁) = (x-x₁)/(x₂-x₁)已知两点
截距式x/a + y/b = 1已知x,y轴截距
一般式Ax+By+C = 0通用形式

一般式的几何意义：

  • 斜率k = -A/B（B≠0）
  • 法向量n⃗ = (A, B)（垂直于直线的向量）

2.2 两直线的位置关系 ​

设l₁: A₁x+B₁y+C₁=0，l₂: A₂x+B₂y+C₂=0

位置关系条件
平行A₁B₂ = A₂B₁ 且 B₁C₂ ≠ B₂C₁
垂直A₁A₂ + B₁B₂ = 0（斜率乘积为-1）
相交A₁B₂ ≠ A₂B₁
重合A₁/A₂ = B₁/B₂ = C₁/C₂

点到直线距离：

d = |Ax₀+By₀+C| / √(A²+B²)

2.3 圆的方程 ⭐ ​

标准方程：

(x-a)² + (y-b)² = r²
  • 圆心(a, b)，半径r

一般方程：

x² + y² + Dx + Ey + F = 0
  • 配方得标准形式：(x+D/2)² + (y+E/2)² = (D²+E²-4F)/4
  • 当D²+E²-4F > 0时表示圆

直线与圆的位置关系：

  • 几何法：比较圆心到直线距离d与半径r
    • d < r：相交（2个交点）
    • d = r：相切（1个交点）
    • d > r：相离（0个交点）
  • 代数法：联立方程，看判别式Δ

2.4 椭圆 ⭐ ​

定义：平面内到两定点F₁, F₂的距离之和为常数2a的点的轨迹

  • 2a > |F₁F₂| = 2c（否则不是椭圆）

标准方程：

  • 焦点在x轴：x²/a² + y²/b² = 1（a > b > 0）
  • 焦点在y轴：y²/a² + x²/b² = 1（a > b > 0）

参数关系：

  • a：半长轴，b：半短轴，c：半焦距
  • c² = a² - b²
  • 离心率：e = c/a（0 < e < 1）

几何性质：

  • 范围：|x|≤a, |y|≤b（焦点在x轴时）
  • 对称性：关于x轴、y轴、原点对称
  • 顶点：(±a, 0), (0, ±b)

2.5 双曲线 ​

定义：平面内到两定点F₁, F₂的距离之差的绝对值为常数2a的点的轨迹

  • 2a < |F₁F₂| = 2c

标准方程：

  • 焦点在x轴：x²/a² - y²/b² = 1
  • 焦点在y轴：y²/a² - x²/b² = 1

参数关系：

  • c² = a² + b²（注意是+号，不是-号）
  • 离心率：e = c/a（e > 1）
  • 渐近线：y = ±(b/a)x（焦点在x轴时）

2.6 抛物线 ⭐ ​

定义：平面内到一定点F（焦点）和定直线l（准线）距离相等的点的轨迹

标准方程（四种形式）：

方程焦点准线开口方向
y² = 2px (p>0)(p/2, 0)x = -p/2向右
y² = -2px (p>0)(-p/2, 0)x = p/2向左
x² = 2py (p>0)(0, p/2)y = -p/2向上
x² = -2py (p>0)(0, -p/2)y = p/2向下

核心性质：抛物线上任一点到焦点的距离等于到准线的距离

2.7 直线与圆锥曲线的位置关系 ⭐ ​

通用方法：

  1. 设直线方程：y = kx + m 或 x = my + n
  2. 联立圆锥曲线方程
  3. 消元得一元二次方程
  4. 用判别式Δ判断交点个数
  5. 用韦达定理求交点关系

弦长公式：

|AB| = √(1+k²) · |x₁-x₂| = √(1+k²) · √Δ/|a|

设点设线策略：

  • 设点：用参数表示曲线上一点（如椭圆上的点(acosθ, bsinθ)）
  • 设线：y = kx + m（斜率存在时）或x = my + n（斜率不存在或需简化计算时）

三、典型例题 ​

例题1：求圆的方程 ​

题目：求过点A(1,2), B(3,4)，圆心在直线x-y+1=0上的圆的方程。

解答：

设圆心(a,b)，则：
① (a-1)²+(b-2)² = (a-3)²+(b-4)²  【到A、B距离相等】
② a-b+1 = 0  【圆心在给定直线上】

由①：展开化简得 a+b=5
由②：a-b=-1

解得：a=2, b=3
半径r = √[(2-1)²+(3-2)²] = √2

∴ 圆的方程：(x-2)²+(y-3)² = 2

例题2：椭圆方程与性质 ​

题目：椭圆中心在原点，焦点在x轴，离心率e=1/2，且过点(2,3)，求椭圆方程。

解答：

设椭圆方程：x²/a² + y²/b² = 1

已知：
① e = c/a = 1/2 → c = a/2
② c² = a² - b² → a²/4 = a² - b² → b² = 3a²/4
③ 过点(2,3)：4/a² + 9/b² = 1

将b²=3a²/4代入③：
4/a² + 9/(3a²/4) = 1
4/a² + 12/a² = 1
16/a² = 1
∴ a²=16, b²=12

椭圆方程：x²/16 + y²/12 = 1

例题3：直线与椭圆相交 ​

题目：直线y=x+1与椭圆x²/4+y²=1相交于A,B两点，求弦长|AB|。

解答：

联立：
x²/4 + (x+1)² = 1
x²/4 + x²+2x+1 = 1
5x²/4 + 2x = 0
5x² + 8x = 0
x(5x+8) = 0

∴ x₁=0, x₂=-8/5
对应y₁=1, y₂=-3/5

弦长|AB| = √[(x₂-x₁)²+(y₂-y₁)²]
         = √[(64/25)+(64/25)]
         = √(128/25) = 8√2/5

或用弦长公式：

联立后：5x²+8x=0
Δ = 64
|AB| = √(1+1)·√64/5 = √2·8/5 = 8√2/5

四、配套练习 ​

基础练习（必做） ​

练习1：求过点(1,2)和(3,0)的直线方程。

练习2：求圆心为(2,-1)，且与直线3x-4y+5=0相切的圆的方程。

练习3：椭圆x²/25+y²/9=1的焦点坐标是？离心率是？

练习4：抛物线y²=8x的焦点坐标是？准线方程是？

参考答案：

  • 练习1：x+y-3=0（或y=-x+3）
  • 练习2：(x-2)²+(y+1)²=4（距离d=|6+4+5|/5=3，半径应为3？重新算：d=|6+4+5|/5=3，方程为(x-2)²+(y+1)²=9）
  • 练习3：焦点(±4,0)，离心率e=4/5
  • 练习4：焦点(2,0)，准线x=-2

提高练习（选做） ​

练习5：椭圆x²/a²+y²/b²=1(a>b>0)的离心率为√3/2，且过点(2,1)，求椭圆方程。

练习6：直线y=kx+1与双曲线x²-y²=1交于A,B两点，若|AB|=4√2，求k的值。

五、FAQ常见问题 ​

FAQ 1：椭圆和双曲线的a,b,c关系总记混 ​

记忆方法：

  • 椭圆：a最大，c²=a²-b²（减号）
  • 双曲线：c最大，c²=a²+b²（加号）

口诀：

  • 椭圆：a是老大，c²=a²-b²
  • 双曲线：c是老大，c²=a²+b²

FAQ 2：抛物线方程四种形式总搞混 ​

判断方法：

  • 看x²还是y²：x²→开口上下，y²→开口左右
  • 看系数正负：正→向正方向，负→向负方向

示例：

  • y²=4x：y²→左右，正→向右
  • x²=-8y：x²→上下，负→向下

FAQ 3：直线与圆锥曲线联立计算量大 ​

简化技巧：

  1. 设线方式：若过x轴上定点，设x=my+n可能更简单
  2. 韦达定理：不求具体根，用x₁+x₂和x₁x₂表示
  3. 整体代入：弦长、面积等常用整体式表示

FAQ 4：怎么判断设点还是设线 ​

设点：

  • 问题涉及曲线上一个动点
  • 用参数表示（如椭圆(acosθ,bsinθ)）

设线：

  • 问题涉及直线与曲线相交
  • 联立方程求交点关系

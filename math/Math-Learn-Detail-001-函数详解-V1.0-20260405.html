二、概念定义 ​

2.1 函数的概念 ​

一句话定义：函数是非空数集之间的对应关系，满足"每个输入有唯一输出"。

三要素：

  1. 定义域：自变量x的取值范围
  2. 值域：因变量y的取值范围
  3. 对应法则f：x→y的对应规则

关键理解：

  • 函数是"关系"，不是"数"
  • 判断是否为函数：看一个x是否对应唯一y（图像：垂直线检验）

示例：

  • y = 2x + 1 是函数（线性对应）
  • x² + y² = 1 不是函数（一个x对应两个y）

2.2 函数的单调性 ⭐ ​

定义：

  • 增函数：x₁ < x₂ 时，f(x₁) < f(x₂)（图像上升）
  • 减函数：x₁ < x₂ 时，f(x₁) > f(x₂)（图像下降）

判断方法：

  1. 定义法：任取x₁ < x₂，比较f(x₁)与f(x₂)
  2. 图像法：观察图像上升/下降趋势
  3. 导数法（高二学）：f'(x) > 0 → 增；f'(x) < 0 → 减

应用：

  • 求最值：先找单调区间，再比较端点
  • 解不等式：利用单调性去掉f

2.3 函数的奇偶性 ⭐ ​

定义：

  • 偶函数：f(-x) = f(x)（图像关于y轴对称）
  • 奇函数：f(-x) = -f(x)（图像关于原点对称）

判断步骤：

  1. 先看定义域是否关于原点对称（必要条件）
  2. 计算f(-x)
  3. 比较f(-x)与f(x)、-f(x)的关系

性质：

  • 偶 + 偶 = 偶；奇 + 奇 = 奇
  • 偶 × 偶 = 偶；奇 × 奇 = 偶；奇 × 偶 = 奇

2.4 指数与对数 ​

指数运算（a > 0，m,n为实数）：

  • aᵐ · aⁿ = aᵐ⁺ⁿ
  • (aᵐ)ⁿ = aᵐⁿ
  • (ab)ⁿ = aⁿbⁿ

对数定义：

  • 若 aˣ = N（a > 0，a ≠ 1），则 x = logₐN
  • 常用对数：lg N = log₁₀N
  • 自然对数：ln N = logₑN（e ≈ 2.718）

对数运算：

  • logₐ(MN) = logₐM + logₐN
  • logₐ(M/N) = logₐM - logₐN
  • logₐMⁿ = n·logₐM
  • 换底公式：logₐb = logₙb / logₙa

指数函数 y = aˣ（a > 0，a ≠ 1）：

  • a > 1：单调递增
  • 0 < a < 1：单调递减
  • 过定点(0, 1)

对数函数 y = logₐx（a > 0，a ≠ 1）：

  • a > 1：单调递增
  • 0 < a < 1：单调递减
  • 过定点(1, 0)
  • 与指数函数互为反函数

2.5 函数零点 ⭐ ​

定义：f(x) = 0 的解，即函数图像与x轴交点的横坐标。

零点存在定理：

  • 若f(x)在[a,b]上连续，且f(a)·f(b) < 0
  • 则(a,b)内至少有一个零点

求零点方法：

  1. 代数法：解方程f(x) = 0
  2. 图像法：画y=f(x)看与x轴交点
  3. 二分法：利用零点存在定理逐步逼近

三、典型例题 ​

例题1：求函数定义域 ​

题目：求f(x) = √(x+1) + ln(2-x)的定义域

解答：

要使f(x)有意义，需要：
① x + 1 ≥ 0  →  x ≥ -1
② 2 - x > 0  →  x < 2

取交集：-1 ≤ x < 2
∴ 定义域为[-1, 2)

关键点：

  • 偶次根号下≥0
  • 对数真数>0
  • 分母≠0（本题无分母）

例题2：判断函数奇偶性 ​

题目：判断f(x) = x³ - 2x的奇偶性

解答：

第一步：看定义域
定义域为R，关于原点对称 ✓

第二步：计算f(-x)
f(-x) = (-x)³ - 2(-x) = -x³ + 2x = -(x³ - 2x) = -f(x)

第三步：判断
∵ f(-x) = -f(x)
∴ f(x)是奇函数

关键点：

  • 先看定义域是否对称
  • 多项式函数：偶次幂→偶，奇次幂→奇

例题3：指数对数运算 ​

题目：计算 (1/4)⁻² + log₂3 · log₃4

解答：

(1/4)⁻² = 4² = 16

log₂3 · log₃4 = log₂3 · (log₂4 / log₂3)  【换底公式】
               = log₂4
               = 2

∴ 原式 = 16 + 2 = 18

关键点：

  • 负指数：a⁻ⁿ = 1/aⁿ
  • 换底公式：logₐb · log_bc = logₐc（链式法则）

练习2：判断下列函数的奇偶性

  • (1) f(x) = x⁴ - 2x² + 1
  • (2) f(x) = x + 1/x

练习3：计算

  • (1) 8^(2/3) + log₂6 - log₂3
  • (2) log₃5 · log₅9

参考答案：

  • 练习1：(1) [1/2, 3)∪(3, +∞)；(2) (-2, 4]
  • 练习2：(1) 偶函数；(2) 奇函数

五、FAQ常见问题 ​

FAQ 1：我求定义域时总是漏条件 ​

错误示例：

  • 只考虑根号下≥0，忘了分母≠0
  • 只考虑对数真数>0，忘了根号下≥0

正确做法：

  • 建立检查清单：看到根号→≥0；看到对数→>0；看到分母→≠0
  • 所有条件取交集

FAQ 2：指数对数运算总出错 ​

常见错误：

  • (aᵐ)ⁿ ≠ a^(mⁿ) 而是 a^(m·n)
  • logₐ(M+N) ≠ logₐM + logₐN（对数没有分配律）

记忆技巧：

  • 指数：相乘→指数相加；幂的幂→指数相乘
  • 对数：真数相乘→对数相加；真数相除→对数相减

FAQ 3：判断奇偶性时总忘看定义域 ​

错误示例： f(x) = x²，x∈[0, +∞)，说它是偶函数

正确做法：

  • 先看定义域是否关于原点对称（即若x在定义域，-x也在）
  • [0, +∞)不对称，所以非奇非偶

FAQ 4：对数换底公式记不住 ​

推导帮助记忆：

设 logₐb = x，则 aˣ = b
两边取以n为底的对数：logₙ(aˣ) = logₙb
x · logₙa = logₙb
x = logₙb / logₙa
∴ logₐb = logₙb / logₙa

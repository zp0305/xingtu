解析几何套路 ​

设点/设线→联立→韦达→翻译条件

一、题型识别 ​

特征关键词 ​

  • "椭圆/双曲线/抛物线"
  • "直线与圆锥曲线相交"
  • "弦长"
  • "面积"
  • "定点/定值"
  • "中点"

典型问法 ​

  • 求椭圆/抛物线的方程
  • 证明直线过定点
  • 求弦长的取值范围
  • 求三角形面积的最大值
  • 证明某值为定值

二、解题步骤 ​

类型1：求圆锥曲线方程 ​

Step 1：设方程

  • 椭圆：x²/a² + y²/b² = 1（焦点在x轴）
  • 抛物线：y² = 2px（开口向右）

Step 2：列条件

  • 已知点代入
  • 已知离心率e = c/a
  • 已知焦点位置

Step 3：解方程组

  • 利用a、b、c关系（椭圆：c²=a²-b²）

类型2：直线与圆锥曲线相交（核心套路） ​

Step 1：设直线方程

选择1：y = kx + m

  • 适用：斜率存在，且不为0
  • 注意：讨论斜率不存在的情况

选择2：x = my + n

  • 适用：直线过x轴上定点，或需简化计算
  • 避免斜率不存在的讨论

Step 2：联立方程

  • 将直线方程代入圆锥曲线方程
  • 消元得一元二次方程

Step 3：韦达定理

  • 设交点A(x₁,y₁)、B(x₂,y₂)
  • x₁+x₂ = ...，x₁x₂ = ...
  • 判别式Δ > 0（保证相交）

Step 4：翻译条件

  • 弦长：|AB| = √(1+k²)·|x₁-x₂|
  • 中点：((x₁+x₂)/2, (y₁+y₂)/2)
  • 斜率：k_AB = (y₂-y₁)/(x₂-x₁)

Step 5：化简求解

类型3：弦长问题 ​

公式：

|AB| = √(1+k²) · |x₁-x₂|
     = √(1+k²) · √[(x₁+x₂)² - 4x₁x₂]
     = √(1+k²) · √Δ/|a|

步骤：

  1. 联立直线与曲线
  2. 用韦达定理表示x₁+x₂和x₁x₂
  3. 代入弦长公式
  4. 化简，求最值或范围

类型4：面积问题 ​

类型4.1：三角形面积

公式1（底×高）：

S = (1/2) · |AB| · d
  • |AB|：弦长
  • d：点到直线的距离

公式2（坐标公式）：

S = (1/2)|x₁y₂ - x₂y₁|

步骤：

  1. 求弦长|AB|
  2. 求点C到直线AB的距离d
  3. S = (1/2)|AB|·d
  4. 化为单变量函数，求最值

类型4.2：四边形面积

  • 对角线垂直：S = (1/2)|AC|·|BD|
  • 一般：分割成三角形

类型5：定点问题 ​

目标：证明直线过定点，或求定点坐标

Step 1：设直线方程

  • y = kx + m 或 x = my + n

Step 2：联立，韦达

Step 3：利用条件

  • 如：OA⊥OB → x₁x₂ + y₁y₂ = 0
  • 如：k₁ + k₂ = 0（斜率互为相反数）

Step 4：化简得关系

  • 得到k与m的关系

Step 5：代入直线方程

  • 整理成y - y₀ = k(x - x₀)形式
  • 定点即为(x₀, y₀)

类型6：定值问题 ​

目标：证明某表达式为定值

Step 1：设变量

  • 设直线斜率k或点的坐标

Step 2：表示目标式

  • 用k表示目标表达式

Step 3：化简

  • 若k消去，则为定值

四、典型例题 ​

例题1：弦长计算 ​

题目：直线y = x + 1与椭圆x²/4 + y² = 1相交于A、B两点，求|AB|。

解答：

解：联立：
{y = x + 1
{x²/4 + y² = 1

代入得：x²/4 + (x+1)² = 1
x²/4 + x² + 2x + 1 = 1
5x²/4 + 2x = 0
x(5x/4 + 2) = 0

∴ x₁ = 0，x₂ = -8/5
y₁ = 1，y₂ = -3/5

|AB| = √[(x₂-x₁)² + (y₂-y₁)²]
     = √[(64/25) + (64/25)]
     = 8√2/5

五、计算技巧 ​

技巧1：设线方式选择 ​

情况设线方式
过y轴上定点(0,m)y = kx + m
过x轴上定点(n,0)x = ty + n（避免斜率不存在讨论）
斜率已知点斜式

技巧2：韦达定理整体代入 ​

不要单独求x₁、x₂，用x₁+x₂和x₁x₂整体表示。

技巧3：硬解定理（选学） ​

对于y = kx + m与x²/a² + y²/b² = 1：

  • x₁+x₂ = -2a²km/(b²+a²k²)
  • x₁x₂ = a²(m²-b²)/(b²+a²k²)
  • 弦长 = 2ab√[(1+k²)(b²+a²k²-m²)]/(b²+a²k²)

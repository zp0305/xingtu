通用格式规范 ​

化学答题的硬性底线——格式不对，扣分很冤

一、化学方程式书写规范 ​

必扣分项（坚决避免） ​

错误类型错误示例正确写法扣分
不配平H₂ + O₂ = H₂O2H₂ + O₂ = 2H₂O1-2分
漏写条件2KClO₃ = 2KCl + 3O₂↑2KClO₃ =△= 2KCl + 3O₂↑0.5-1分
漏写状态符号CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑0.5分
可逆号写错N₂ + 3H₂ = 2NH₃N₂ + 3H₂ ⇌ 2NH₃0.5分
化学式写错NaCO₃Na₂CO₃全扣

状态符号使用规则 ​

符号含义使用场景
↑气体反应物无气体，生成物有气体
↓沉淀溶液中反应生成难溶物
(g)气态热化学方程式
(l)液态热化学方程式
(s)固态热化学方程式
(aq)水溶液热化学方程式、离子方程式

注意：

  • 反应物有气体，生成物气体不标↑
  • 热化学方程式必须注明状态

示例对比 ​

❌ 错误写法：

Cu + 4HNO3 = Cu(NO3)2 + 2NO2 + 2H2O

✅ 正确写法：

Cu + 4HNO₃(浓) =△= Cu(NO₃)₂ + 2NO₂↑ + 2H₂O

检查清单：

  • [ ] 配平了吗？
  • [ ] 反应条件写了吗？
  • [ ] 气体/沉淀符号标了吗？
  • [ ] 化学式写对了吗？

二、离子方程式书写规范 ​

拆与不拆规则 ​

必须拆（写离子）：

  • 强酸：HCl、H₂SO₄、HNO₃、HClO₄等
  • 强碱：NaOH、KOH、Ba(OH)₂、Ca(OH)₂（澄清溶液）
  • 可溶性盐：钠盐、钾盐、铵盐、硝酸盐等

不能拆（写化学式）：

  • 弱酸、弱碱、水
  • 难溶物、单质、氧化物
  • 气体、有机物（除少数有机酸）

常见错误 ​

错误原因正确
CH₃COOH = CH₃COO⁻ + H⁺弱酸不拆CH₃COOH ⇌ CH₃COO⁻ + H⁺
CaCO₃ = Ca²⁺ + CO₃²⁻难溶物不拆CaCO₃（保留化学式）
NH₃·H₂O = NH₄⁺ + OH⁻弱碱不拆NH₃·H₂O ⇌ NH₄⁺ + OH⁻

检查清单 ​

  • [ ] 强酸强碱可溶性盐拆了吗？
  • [ ] 弱酸弱碱难溶物没拆吗？
  • [ ] 原子守恒吗？
  • [ ] 电荷守恒吗？

三、热化学方程式规范 ​

必写要素 ​

  1. 注明物质状态：(g)、(l)、(s)、(aq)
  2. 注明ΔH：+ 吸热，- 放热
  3. 单位：kJ/mol
  4. 化学计量数与ΔH对应

示例 ​

H₂(g) + ½O₂(g) = H₂O(l)  ΔH = -285.8 kJ/mol
2H₂(g) + O₂(g) = 2H₂O(l)  ΔH = -571.6 kJ/mol

注意：

  • ΔH与化学计量数成正比
  • 逆反应的ΔH符号相反

四、电极反应式规范 ​

书写步骤 ​

  1. 确定电极：阳极/阴极（或正极/负极）
  2. 确定反应物：电极材料或溶液中离子
  3. 写出产物
  4. 配平：
    • 原子守恒（先用H₂O、H⁺、OH⁻配平）
    • 电荷守恒（用e⁻配平）

示例 ​

电解NaCl溶液：

阳极：2Cl⁻ - 2e⁻ = Cl₂↑
阴极：2H⁺ + 2e⁻ = H₂↑  （或 2H₂O + 2e⁻ = H₂↑ + 2OH⁻）

注意：

  • 酸性溶液用H⁺配平
  • 碱性溶液用OH⁻配平
  • 必须注明电子得失（- 失电子，+ 得电子）

五、计算题规范 ​

步骤分很重要 ​

❌ 错误示范：

9 g水的物质的量是0.5 mol

✅ 正确示范：

解：
M(H₂O) = 18 g/mol
n(H₂O) = m/M = 9 g / 18 g/mol = 0.5 mol
答：9 g水的物质的量是0.5 mol

必须写出的要素 ​

  1. 已知条件
  2. 所用公式
  3. 代入数据（带单位）
  4. 计算结果（带单位）
  5. 答（完整回答）

有效数字 ​

  • 一般保留2-3位有效数字
  • 题目有要求按题目要求
  • 不要无限位小数（如0.500000000）

六、简答题规范 ​

答题模板 ​

问：为什么...？

答：因为...（原理），导致...（结果），所以...（结论）

问：有什么现象？

答：...（颜色变化），...（沉淀/气体），...（温度变化）

问：如何操作？

答：第一步：...；第二步：...；第三步：...（分点作答）

示例 ​

问：为什么收集氨气要用向下排空气法？

❌ 错误答法：因为氨气密度小。

✅ 正确答法：

答：因为NH₃的相对分子质量为17，小于空气的平均相对分子质量29，
    所以NH₃密度小于空气，且NH₃极易溶于水，不能用排水法收集，
    故采用向下排空气法收集。

七、实验题规范 ​

现象描述要点 ​

必须包含：

  1. 视觉：颜色变化、沉淀生成、气体产生
  2. 听觉：声音（如爆鸣声）
  3. 嗅觉：气味（题目要求时）
  4. 触觉：温度变化（放热/吸热）

示例 ​

钠与水反应：

现象：钠浮在水面上（浮），熔成闪亮的小球（熔），
      四处游动（游），发出嘶嘶的响声（响），
      溶液变红（加酚酞后）（红）

操作描述要点 ​

  • 用药匙取用固体
  • 用镊子取用块状固体
  • 滴管垂直悬空，不伸入容器
  • 倾倒液体时标签向手心

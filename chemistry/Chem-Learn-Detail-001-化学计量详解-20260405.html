化学计量详解 ​

板块1：化学计算的基石

物质的量是高中化学的「硬门槛」，计算不过关，后续全崩。

第一章：物质的量（摩尔） ​

1.1 核心概念 ​

物质的量是表示含有一定数目粒子的集合体的物理量，单位是摩尔（mol）。

为什么要引入物质的量？

  • 微观粒子（原子、分子）太小，无法直接计数
  • 宏观质量可以测量，但与粒子数不直观对应
  • 物质的量是宏观质量与微观粒子数之间的「桥梁」

阿伏伽德罗常数：

  • 1 mol任何粒子所含的粒子数约为 6.02×10²³
  • 符号：Nₐ

计算公式：

n = N/Nₐ

n：物质的量（mol）
N：粒子数目
Nₐ：阿伏伽德罗常数（6.02×10²³ mol⁻¹）

【典型例题1】 3.01×10²³个水分子是多少摩尔？

【详解】 n = N/Nₐ = 3.01×10²³ / 6.02×10²³ = 0.5 mol

1.2 摩尔质量 ​

定义：单位物质的量的物质所具有的质量，单位 g/mol。

规律：

  • 数值上等于相对原子质量或相对分子质量
  • 单位是 g/mol，不是 g

计算公式：

n = m/M

n：物质的量（mol）
m：质量（g）
M：摩尔质量（g/mol）

常见物质的摩尔质量：

物质化学式摩尔质量(g/mol)
水H₂O18
二氧化碳CO₂44
氢氧化钠NaOH40
硫酸H₂SO₄98
氯化钠NaCl58.5

【典型例题2】 9 g水（H₂O）的物质的量是多少？

【详解】

  • M(H₂O) = 18 g/mol
  • n = m/M = 9 g / 18 g/mol = 0.5 mol

1.3 气体摩尔体积 ​

定义：单位物质的量的气体所占的体积，标准状况下约为 22.4 L/mol。

⚠️ 重要说明（易错点）：

  • 标准状况：0℃（273K），101 kPa
  • 只适用于气态物质
  • 液态和固态物质不能用 22.4 L/mol

计算公式：

n = V/Vm

n：物质的量（mol）
V：气体体积（L）
Vm：气体摩尔体积（标准状况下 22.4 L/mol）

【典型例题3】 标准状况下，11.2 L CO₂的物质的量是多少？质量是多少？

【详解】

  • n = V/Vm = 11.2 L / 22.4 L/mol = 0.5 mol
  • m = n × M = 0.5 mol × 44 g/mol = 22 g

1.4 物质的量浓度 ​

定义：单位体积溶液中所含溶质的物质的量，单位 mol/L。

计算公式：

c = n/V  或  n = cV

n：溶质的物质的量（mol）
c：物质的量浓度（mol/L）
V：溶液体积（L）

⚠️ 重要提醒：

  • V是溶液体积，不是溶剂体积
  • 单位换算：1 L = 1000 mL

【典型例题4】 配制 500 mL 0.1 mol/L NaOH溶液，需要NaOH多少克？

【详解】

  • V = 0.5 L
  • n = cV = 0.1 mol/L × 0.5 L = 0.05 mol
  • m = nM = 0.05 mol × 40 g/mol = 2 g

第二章：化学计量综合计算 ​

2.1 「桥梁」公式总结 ​

                    n（物质的量）
                   ↗    ↓    ↘
              N/Nₐ     cV     m/M
                 ↓      ↓       ↓
              粒子数  浓度×体积  质量/摩尔质量
                      （气：V/Vm）

核心思想：

  • 无论已知什么条件，先转化为 n
  • 再根据化学方程式进行计算
  • 最后转化为要求的物理量

2.2 标准解题步骤 ​

Step 1：找出已知条件

  • 已知质量 → 用 n = m/M
  • 已知气体体积（标况）→ 用 n = V/Vm
  • 已知浓度和体积 → 用 n = cV

Step 2：转化为物质的量 n

Step 3：根据化学方程式计算

  • 利用化学计量数之比 = 物质的量之比

Step 4：转化为要求的物理量

2.3 典型例题 ​

【综合例题】 标准状况下 11.2 L HCl气体溶于水配成 500 mL溶液。 (1) 求所得溶液的物质的量浓度。 (2) 取该溶液 100 mL，能与多少克 NaOH完全反应？

【详解】

(1) 求浓度

  • n(HCl) = V/Vm = 11.2 L / 22.4 L/mol = 0.5 mol
  • c = n/V = 0.5 mol / 0.5 L = 1 mol/L

(2) 求反应的NaOH质量

  • 取 100 mL溶液，n(HCl) = cV = 1 mol/L × 0.1 L = 0.1 mol
  • 反应：HCl + NaOH = NaCl + H₂O
  • 物质的量之比 1:1，n(NaOH) = 0.1 mol
  • m(NaOH) = nM = 0.1 mol × 40 g/mol = 4 g

第三章：易错点警示 ​

⚠️ 易错点1：22.4 L/mol的适用条件 ​

错误说法错误原因正确说法
"1 mol水在标况下体积是22.4 L"水是液态，不适用22.4 L/mol只适用于气体
"常温常压下22.4 L氧气是1 mol"不是标准状况必须是标况（0℃，101 kPa）

⚠️ 易错点2：单位换算 ​

易错点正确做法
体积用 mL 直接计算必须先转化为 L：V(L) = V(mL) / 1000
摩尔质量单位写成 g单位是 g/mol

⚠️ 易错点3：溶液体积 vs 溶剂体积 ​

  • ❌ 错误：把溶剂体积当溶液体积
  • ✅ 正确：题目说「配成 500 mL溶液」，V = 0.5 L

第四章：配套练习 ​

基础练习 ​

1. 3.01×10²³个氧分子是多少摩尔？质量是多少克？

2. 标准状况下，5.6 L CO₂的质量是多少克？

3. 配制 250 mL 0.2 mol/L H₂SO₄溶液，需要H₂SO₄多少克？

提高练习 ​

4. 标准状况下，11.2 L某气体的质量为14 g，求该气体的相对分子质量。

5. 将 4 g NaOH溶于水配成 500 mL溶液。 (1) 求溶液的物质的量浓度。 (2) 取该溶液 100 mL，加水稀释至 500 mL，求稀释后溶液的浓度。

参考答案 ​

1. n = 0.5 mol，m = 16 g

2. n = 0.25 mol，m = 11 g

3. n = 0.05 mol，m = 4.9 g

4. n = 0.5 mol，M = 28 g/mol，相对分子质量 = 28

5. (1) c = 0.2 mol/L (2) 稀释后 c = 0.04 mol/L

FAQ（常见问题） ​

Q1：物质的量计算时单位总是搞混怎么办？ ​

A：记住「物质的量 n」是核心：

  • n = m/M（质量÷摩尔质量）
  • n = V/Vm（气体体积÷22.4 L/mol）
  • n = cV（浓度×体积，注意体积单位是L）

检查方法：算完后看单位是否合理（n 的单位必须是 mol）。

Q2：不知道该用哪个公式怎么办？ ​

A：看已知条件：

已知条件选用公式
质量和摩尔质量n = m/M
标况下气体体积n = V/Vm
浓度和溶液体积n = cV
粒子数n = N/Nₐ

Q3：气体摩尔体积什么时候不能用？ ​

A：以下情况不能用 22.4 L/mol：

  • 物质不是气体（如水、酒精在标况下是液体）
  • 不是标准状况（常温常压、高温高压等）
  • SO₃在标况下是固体

有机化学基础详解 ​

板块4：烃类 + 烃的衍生物 + 有机推断（选考必考）

学习策略 ​

有机化学的核心是「官能团」：

  • 官能团决定性质
  • 性质决定反应
  • 反应决定转化关系

学习方法：

  1. 熟记官能团：名称、结构、性质、特征反应
  2. 掌握转化关系：醇→醛→酸→酯的转化
  3. 找突破口：推断题的特征现象、分子式、反应条件
  4. 多画图：官能团转化关系网、同分异构体树状图

第一章：烃类 ​

1.1 烷烃 ​

通式：CₙH₂ₙ₊₂

代表物：甲烷（CH₄）

化学性质（稳定，取代反应为主）：

CH₄ + Cl₂ → CH₃Cl + HCl（光照条件，连锁反应）

特点：

  • 不能使溴水褪色
  • 不能使KMnO₄褪色
  • 取代反应，产物混合物

1.2 烯烃 ​

通式：CₙH₂ₙ

代表物：乙烯（CH₂=CH₂）

官能团：碳碳双键（C=C）

化学性质（活泼，加成反应为主）：

1. 加成反应：

CH₂=CH₂ + Br₂ → CH₂Br-CH₂Br（使溴水褪色）
CH₂=CH₂ + H₂ → CH₃-CH₃（催化加氢）
CH₂=CH₂ + H₂O → CH₃-CH₂OH（水化制乙醇）

2. 氧化反应：

CH₂=CH₂ + 3O₂ → 2CO₂ + 2H₂O（燃烧）
5CH₂=CH₂ + 12KMnO₄ + 18H₂SO₄ → ...（使KMnO₄褪色）

3. 加聚反应：

nCH₂=CH₂ → [CH₂-CH₂]n（聚乙烯）

特征反应：能使溴水、KMnO₄溶液褪色（检验烯烃）

1.3 炔烃 ​

通式：CₙH₂ₙ₋₂

代表物：乙炔（CH≡CH）

官能团：碳碳三键（C≡C）

化学性质：与烯烃类似，能加成、氧化、加聚

1.4 芳香烃 ​

代表物：苯（C₆H₆）

结构特点：平面正六边形，碳碳键介于单双键之间

化学性质（稳定，特殊结构）：

1. 取代反应（不是加成）：

C₆H₆ + Br₂ → C₆H₅Br + HBr（FeBr₃催化）
C₆H₆ + HNO₃ → C₆H₅NO₂ + H₂O（浓H₂SO₄，加热）

2. 加成反应（需催化剂、加热）：

C₆H₆ + 3H₂ → C₆H₁₂（环己烷，Ni催化）

注意：苯不能使溴水褪色（萃取除外），不能使KMnO₄褪色。

第二章：烃的衍生物 ​

2.1 官能团速查表 ​

类别官能团代表物特征反应
卤代烃-X（Cl、Br）CH₃CH₂Cl水解、消去
醇-OH（羟基）CH₃CH₂OH与Na反应、氧化、酯化
酚-OH（直接连苯环）C₆H₅OH弱酸性、与FeCl₃显色
醛-CHO（醛基）CH₃CHO银镜反应、与新制Cu(OH)₂反应
酮-CO-（羰基）CH₃COCH₃加成反应
羧酸-COOH（羧基）CH₃COOH酸性、酯化
酯-COO-（酯基）CH₃COOCH₂CH₃水解

2.2 醇（重点） ​

代表物：乙醇（CH₃CH₂OH）

化学性质：

1. 与活泼金属反应（如Na）：

2CH₃CH₂OH + 2Na → 2CH₃CH₂ONa + H₂↑

2. 氧化反应：

2CH₃CH₂OH + O₂ → 2CH₃CHO + 2H₂O（Cu/Ag催化，生成醛）

3. 消去反应：

CH₃CH₂OH → CH₂=CH₂ + H₂O（浓H₂SO₄，170℃）

4. 酯化反应：

CH₃COOH + CH₃CH₂OH ⇌ CH₃COOCH₂CH₃ + H₂O（浓H₂SO₄，加热）

2.3 醛（重点，检验特征明显） ​

代表物：乙醛（CH₃CHO）

特征反应（检验醛基）：

1. 银镜反应（与银氨溶液）：

CH₃CHO + 2[Ag(NH₃)₂]OH → CH₃COONH₄ + 2Ag↓ + 3NH₃ + H₂O
  • 现象：试管内壁形成光亮银镜
  • 条件：水浴加热

2. 与新制Cu(OH)₂反应：

CH₃CHO + 2Cu(OH)₂ + NaOH → CH₃COONa + Cu₂O↓ + 3H₂O
  • 现象：生成砖红色沉淀（Cu₂O）
  • 条件：加热至沸腾

2.4 羧酸和酯 ​

羧酸代表物：乙酸（CH₃COOH，醋酸）

性质：

  • 弱酸性：CH₃COOH ⇌ CH₃COO⁻ + H⁺
  • 酯化反应（见上文）

酯代表物：乙酸乙酯（CH₃COOCH₂CH₃）

性质：水解反应

CH₃COOCH₂CH₃ + H₂O ⇌ CH₃COOH + CH₃CH₂OH（酸性条件）
CH₃COOCH₂CH₃ + NaOH → CH₃COONa + CH₃CH₂OH（碱性条件，皂化）

第三章：有机反应类型总结 ​

3.1 反应类型速查 ​

反应类型特点典型例子
取代反应原子或基团被替代烷烃卤代、苯的硝化、酯化、水解
加成反应不饱和键断裂，加上原子或基团烯烃加Br₂、加H₂、加H₂O
消去反应从分子中消去小分子（H₂O、HX）醇消去生成烯烃
氧化反应加氧或去氢醇→醛→酸、烯烃使KMnO₄褪色
还原反应加氢或去氧醛→醇、烯烃加氢
加聚反应不饱和化合物聚合成高分子乙烯→聚乙烯
缩聚反应单体间脱去小分子聚合氨基酸→蛋白质

3.2 反应条件与反应类型对应 ​

反应条件可能的反应类型
光照烷烃卤代（取代）
浓H₂SO₄，170℃醇消去
浓H₂SO₄，加热酯化、硝化、磺化（取代）
NaOH水溶液，加热卤代烃水解、酯水解
NaOH醇溶液，加热卤代烃消去
Cu/Ag，加热醇氧化成醛
Ni，加热烯烃/醛/酮加氢（还原）
银氨溶液，水浴醛基氧化（银镜反应）
新制Cu(OH)₂，加热醛基氧化

第四章：有机推断题攻略 ​

4.1 突破口总结 ​

1. 特征现象突破口

现象推断
使溴水褪色含C=C、C≡C或醛基
使KMnO₄褪色含C=C、C≡C、醛基或苯的同系物
银镜反应含醛基（-CHO）
与FeCl₃显紫色含酚羟基
与Na反应放出H₂含-OH或-COOH
与Na₂CO₃/NaHCO₃反应放出CO₂含-COOH

2. 分子式突破口

  • 计算不饱和度：Ω = (2C + 2 + N - H - X)/2
  • Ω=0：饱和化合物（烷烃、醇、醚）
  • Ω=1：含一个双键或一个环（烯烃、酮、醛）
  • Ω=2：含两个双键或一个三键或一个双键+一个环
  • Ω=4：可能含苯环

3. 反应条件突破口

  • 见上文「反应条件与反应类型对应」

4. 转化关系突破口

常见转化链：

醇 → 醛 → 羧酸 → 酯
↑              ↓
└── 水解 ────────┘

4.2 有机推断题解题步骤 ​

  1. 审题：找出分子式、反应条件、特征现象
  2. 算不饱和度：推测可能结构
  3. 找突破口：从特征现象或条件入手
  4. 顺藤摸瓜：根据转化关系推导其他物质
  5. 验证：检查是否符合所有条件

4.3 同分异构体书写 ​

步骤：

  1. 写碳链异构：主链由长到短
  2. 写位置异构：官能团位置变化
  3. 写官能团异构：不同官能团（如醇和醚）

例子：C₄H₁₀O的同分异构体（醇和醚）

  • 醇：CH₃CH₂CH₂CH₂OH、CH₃CH₂CH(OH)CH₃、(CH₃)₂CHCH₂OH、(CH₃)₃COH
  • 醚：CH₃OCH₂CH₂CH₃、CH₃CH₂OCH₂CH₃、CH₃OCH(CH₃)₂

FAQ（常见错误与解答） ​

Q1：官能团性质太多记不住怎么办？ ​

A：重点记「特征反应」：

  • 能发生银镜反应的：醛基
  • 能使溴水褪色的：C=C、C≡C、醛基
  • 能与Na反应放H₂的：-OH、-COOH
  • 与FeCl₃显色的：酚羟基

其他性质通过官能团推断（如含-OH的能发生酯化）。

Q2：有机推断题总是推不出来怎么办？ ​

A：

  1. 先熟记「转化关系图」（醇-醛-酸-酯）
  2. 记住常见「突破口」（特征现象、分子式、反应条件）
  3. 多做题，总结每道题的突破口是怎么用的
  4. 画推断流程图，不要空想

Q3：同分异构体总是数漏或重复怎么办？ ​

A：

  1. 按步骤来：先碳链异构，再位置异构，最后官能团异构
  2. 用「等效氢法」确定一元取代物数目
  3. 写出来后用系统命名法验证（命名相同就是重复）
  4. 记住常见数目（如C₄H₉-有4种，C₅H₁₁-有8种）

Q4：酯化反应方程式总是写错怎么办？ ​

A：

  • 记住「酸脱羟基，醇脱氢」
  • 可逆号，浓H₂SO₄作催化剂和吸水剂
  • 产物是酯和水，不要漏写水

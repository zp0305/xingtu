化学实验详解 ​

板块5：基本操作 + 分离提纯 + 物质检验 + 实验探究

学习策略 ​

化学实验是「实践检验真理」的体现：

  • 高考实验题源于课本，又高于课本
  • 既考操作，又考原理，还考设计
  • 重视「为什么这样做」，不只是「怎么做」

学习方法：

  1. 熟悉课本实验：高考题都源于课本
  2. 掌握基本操作：分离提纯、气体制备、物质检验
  3. 理解实验原理：每步操作的科学依据
  4. 总结答题模板：实验探究题的回答套路

第一章：化学实验基本操作 ​

1.1 常用仪器 ​

仪器用途注意事项
试管少量试剂反应加热时液体不超过1/3
烧杯较多量试剂反应、溶解加热时垫石棉网
烧瓶加热反应、蒸馏圆底/平底/蒸馏烧瓶区分
锥形瓶滴定、接收液体加热时垫石棉网
蒸发皿蒸发浓缩直接加热，用坩埚钳夹取
坩埚高温灼烧固体配合泥三角、三脚架使用
容量瓶配制一定体积溶液不能加热，不能长期储存
滴定管精确滴加液体酸式/碱式区分，使用前洗涤+润洗
分液漏斗分液、萃取检漏，上层上口出，下层下口出

1.2 基本操作 ​

药品取用：

  • 固体：药匙、镊子（块状用镊子）
  • 液体：倾倒法、滴加法（胶头滴管垂直悬空）
  • 原则：不触（不接触药品）、不闻（不把鼻子凑近闻）、不尝

加热操作：

  • 直接加热：试管、蒸发皿、坩埚
  • 间接加热（垫石棉网）：烧杯、烧瓶、锥形瓶
  • 水浴加热：温度不超过100℃，受热均匀

溶解与搅拌：

  • 固体溶解：搅拌加速，必要时加热
  • 液体稀释：浓硫酸沿器壁慢慢注入水中，不断搅拌

第二章：物质的分离与提纯 ​

2.1 分离方法速查表 ​

方法适用范围仪器关键操作
过滤固液分离漏斗、烧杯、玻璃棒一贴二低三靠
蒸发从溶液中获得固体蒸发皿、酒精灯、玻璃棒不断搅拌，余热蒸干
结晶溶解度随温度变化大的物质烧杯、酒精灯蒸发浓缩、冷却结晶、过滤洗涤
蒸馏沸点不同的液体混合物蒸馏烧瓶、冷凝管、接收器温度计位置，冷水下进上出
萃取溶质在互不相溶的溶剂中溶解度不同分液漏斗充分振荡，静置分层
分液互不相溶的液体分离分液漏斗下层下口出，上层上口出
升华受热易升华的物质（如I₂）烧杯、圆底烧瓶加热使升华，冷却凝华

2.2 过滤操作要点 ​

一贴：滤纸紧贴漏斗内壁 二低：滤纸边缘低于漏斗边缘；液面低于滤纸边缘 三靠：烧杯紧靠玻璃棒；玻璃棒轻靠三层滤纸处；漏斗下端紧靠烧杯内壁

2.3 蒸馏操作要点 ​

温度计位置：水银球位于蒸馏烧瓶支管口处（测蒸气温度） 冷凝水方向：下口进，上口出（逆流原理，冷凝效果好） 加碎瓷片：防止液体暴沸

2.4 萃取与分液 ​

萃取剂选择原则：

  1. 与原溶剂互不相溶
  2. 溶质在萃取剂中的溶解度远大于在原溶剂中
  3. 萃取剂与溶质不发生化学反应

常见萃取例子：

  • 用CCl₄萃取溴水中的Br₂
  • 用苯萃取碘水中的I₂

第三章：常见气体的制备 ​

3.1 气体发生装置 ​

装置类型适用反应例子
固+固，加热制O₂、NH₃KClO₃制O₂、NH₄Cl+Ca(OH)₂制NH₃
固+液，不加热制H₂、CO₂、C₂H₂Zn+H₂SO₄制H₂、CaCO₃+HCl制CO₂
固+液，加热制Cl₂、C₂H₄MnO₂+HCl制Cl₂

3.2 常见气体制备 ​

氧气（O₂）：

2KClO₃ =△= 2KCl + 3O₂↑（MnO₂催化）
2KMnO₄ =△= K₂MnO₄ + MnO₂ + O₂↑
2H₂O₂ =MnO₂= 2H₂O + O₂↑
  • 收集：排水法或向上排空气法

氢气（H₂）：

Zn + H₂SO₄ = ZnSO₄ + H₂↑
  • 收集：排水法或向下排空气法
  • 验纯：点燃前必须验纯

二氧化碳（CO₂）：

CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑
  • 收集：向上排空气法（能溶于水，不能用排水法）
  • 验满：燃着的木条放在瓶口，熄灭则满

氨气（NH₃）：

2NH₄Cl + Ca(OH)₂ =△= CaCl₂ + 2NH₃↑ + 2H₂O
  • 收集：向下排空气法（极易溶于水，密度小于空气）
  • 验满：湿润的红色石蕊试纸放在瓶口，变蓝则满

氯气（Cl₂）：

MnO₂ + 4HCl(浓) =△= MnCl₂ + Cl₂↑ + 2H₂O
  • 收集：向上排空气法或排饱和食盐水法
  • 尾气处理：用NaOH溶液吸收

3.3 气体净化与干燥 ​

除杂原则：不引入新杂质，不减少被净化气体

常见净化装置：

  • 洗气瓶（长进短出）：除去气体中的杂质气体
  • 干燥管/U形管：装固体干燥剂

常见干燥剂：

干燥剂可干燥气体不能干燥气体
浓H₂SO₄H₂、O₂、CO₂、Cl₂、SO₂NH₃（反应）、H₂S（反应）
无水CaCl₂H₂、O₂、N₂、HClNH₃（络合）
碱石灰H₂、O₂、NH₃CO₂、SO₂、Cl₂、HCl（反应）
P₂O₅H₂、O₂、CO₂、HClNH₃（反应）

第四章：物质的检验与鉴别 ​

4.1 阳离子检验 ​

离子检验方法现象
H⁺紫色石蕊试液变红
NH₄⁺加NaOH溶液，加热，湿润红色石蕊试纸检验产生刺激性气味气体，试纸变蓝
Fe²⁺加KSCN溶液，再加氯水先无明显现象，后变血红色
Fe³⁺加KSCN溶液变血红色
Cu²⁺加NaOH溶液产生蓝色沉淀
Ag⁺加稀盐酸或可溶性氯化物产生白色沉淀，不溶于稀HNO₃
Ba²⁺加稀H₂SO₄或可溶性硫酸盐产生白色沉淀，不溶于稀HNO₃

4.2 阴离子检验 ​

离子检验方法现象
OH⁻紫色石蕊试液/酚酞变蓝/变红
Cl⁻加AgNO₃溶液，再加稀HNO₃产生白色沉淀，不溶于稀HNO₃
Br⁻加AgNO₃溶液，再加稀HNO₃产生淡黄色沉淀
I⁻加AgNO₃溶液，再加稀HNO₃产生黄色沉淀
SO₄²⁻先加稀盐酸，再加BaCl₂溶液产生白色沉淀
CO₃²⁻加稀盐酸，气体通入澄清石灰水产生无色无味气体，石灰水变浑浊
SO₃²⁻加稀盐酸，气体通入品红溶液产生刺激性气味气体，品红褪色

4.3 气体检验 ​

气体检验方法现象
O₂带火星的木条木条复燃
H₂点燃，在火焰上方罩冷而干燥的烧杯淡蓝色火焰，烧杯内壁有水珠
CO₂通入澄清石灰水石灰水变浑浊
Cl₂湿润的淀粉-KI试纸试纸变蓝
NH₃湿润的红色石蕊试纸试纸变蓝
SO₂通入品红溶液，加热品红褪色，加热恢复红色

第五章：实验探究题 ​

5.1 实验探究题结构 ​

常见题型：

  1. 假设验证型：给出假设，设计实验验证
  2. 方案评价型：评价给定实验方案的优缺点
  3. 实验设计型：自主设计实验方案

答题模板：

提出假设：

  • 假设一：...
  • 假设二：...
  • 假设三：...

设计实验：

步骤操作预期现象结论
1.........
2.........

得出结论：根据实验现象，支持或否定假设

5.2 实验方案评价角度 ​

科学性：原理是否正确，操作是否可行 安全性：是否有安全隐患 简约性：步骤是否简单，试剂是否易得 经济性：成本是否合理 环保性：是否产生污染，如何处理

5.3 常见答题规范 ​

描述现象要全面：

  • 颜色变化：...色→...色
  • 沉淀：产生...色沉淀
  • 气体：产生...色气体/产生气泡
  • 温度：放热/吸热

解释原因要逻辑清晰：

  1. 发生了什么反应（化学方程式）
  2. 产生了什么物质/变化
  3. 导致什么现象

FAQ（常见错误与解答） ​

Q1：实验操作题总是丢分怎么办？ ​

A：

  1. 熟记「基本操作要点」（如过滤的一贴二低三靠）
  2. 理解每一步操作的目的（为什么这样做）
  3. 注意细节（如温度计位置、冷凝水方向）

Q2：物质检验题总是漏检或错检怎么办？ ​

A：

  1. 熟记「检验顺序」（先检验干扰性强的）
  2. 注意「排除干扰」（如检验Cl⁻前要排除SO₄²⁻干扰）
  3. 记住特征现象（颜色变化、沉淀、气体）

Q3：实验探究题不知道怎么设计实验怎么办？ ​

A：

  1. 使用「对照实验」思想（控制变量）
  2. 使用「假设-验证」框架
  3. 多参考课本实验的设计思路
  4. 记住常见答题模板

Q4：气体制备实验装置连接顺序是什么？ ​

A：

发生装置 → 净化装置 → 干燥装置 → 收集装置 → 尾气处理装置
  • 净化：除去杂质气体
  • 干燥：除去水蒸气
  • 注意：净化和干燥顺序（先除杂后干燥）

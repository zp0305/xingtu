化学反应原理详解 ​

板块3：热化学 + 反应速率 + 化学平衡 + 电化学（高考压轴题核心）

学习警告 ​

本板块是高考化学的「分水岭」：

  • 学得好：化学成绩80分以上
  • 学不好：化学成绩60分以下

核心能力要求：

  • 理解抽象概念（平衡、动态、等效）
  • 掌握图像分析（速率-时间图、转化率-温度图）
  • 熟练计算（平衡常数、转化率、电极反应）

第一章：化学反应与能量（热化学） ​

1.1 反应热与焓变 ​

反应热（ΔH）：化学反应中吸收或放出的热量。

  • ΔH < 0：放热反应
  • ΔH > 0：吸热反应

常见反应类型：

类型例子ΔH
燃烧反应物质燃烧< 0（放热）
中和反应酸碱中和< 0（放热）
大多数化合H₂ + Cl₂ = 2HCl< 0（放热）
大多数分解CaCO₃ = CaO + CO₂> 0（吸热）
电离过程弱电解质电离> 0（吸热）

1.2 热化学方程式 ​

书写规则：

  1. 注明物质状态（s、l、g、aq）
  2. ΔH与化学计量数对应
  3. 放热ΔH为负，吸热ΔH为正

例子：

H₂(g) + ½O₂(g) = H₂O(l)  ΔH = -285.8 kJ/mol
2H₂(g) + O₂(g) = 2H₂O(l)  ΔH = -571.6 kJ/mol

1.3 盖斯定律 ​

内容：化学反应的反应热只与始态和终态有关，与途径无关。

应用：计算难以直接测量的反应热。

例题： 已知： (1) C(s) + O₂(g) = CO₂(g) ΔH₁ = -393.5 kJ/mol (2) CO(g) + ½O₂(g) = CO₂(g) ΔH₂ = -283.0 kJ/mol

求：C(s) + ½O₂(g) = CO(g) 的ΔH

解： (1) - (2) 得： C(s) + ½O₂(g) = CO(g) ΔH = ΔH₁ - ΔH₂ = -393.5 - (-283.0) = -110.5 kJ/mol

第二章：化学反应速率 ​

2.1 反应速率的表示 ​

定义：单位时间内反应物浓度的减少或生成物浓度的增加。

公式：

v = Δc/Δt

单位：mol/(L·s) 或 mol/(L·min)

注意：

  • 速率总是正值
  • 不同物质表示的速率之比 = 化学计量数之比

例子： 对于 N₂ + 3H₂ ⇌ 2NH₃ v(N₂) : v(H₂) : v(NH₃) = 1 : 3 : 2

2.2 影响反应速率的因素 ​

因素影响规律微观解释
浓度浓度增大，速率加快单位体积内分子数增多，碰撞机会增加
压强（气体）压强增大，速率加快相当于浓度增大
温度温度升高，速率加快分子能量增加，活化分子百分数增加
催化剂加催化剂，速率加快降低活化能，活化分子百分数增加
接触面积面积增大，速率加快固体反应物粉碎

重要结论：

  • 温度影响最显著（每升高10℃，速率增大2-4倍）
  • 催化剂只改变速率，不改变平衡

第三章：化学平衡（核心难点） ​

3.1 可逆反应与化学平衡 ​

可逆反应：在同一条件下，既能向正反应方向进行，又能向逆反应方向进行的反应。

化学平衡状态：

  • 正反应速率 = 逆反应速率 ≠ 0（动态平衡）
  • 各物质浓度保持不变

平衡特征：「动、等、定、变」

  • 动：动态平衡
  • 等：正逆速率相等
  • 定：浓度恒定
  • 变：条件改变，平衡移动

3.2 勒夏特列原理 ​

内容：如果改变影响平衡的一个条件（浓度、压强、温度），平衡就向能够减弱这种改变的方向移动。

应用规律：

改变条件平衡移动方向结果
增大反应物浓度正向移动反应物浓度减小（但不能回到原值）
减小生成物浓度正向移动生成物浓度增大
增大压强（气体）向气体分子数少的方向压强减小
升高温度向吸热方向温度降低
加催化剂不移动正逆速率同倍数增加

口诀：「增反减生正移，增压向少，升温向吸」

3.3 化学平衡常数（K） ​

表达式： 对于 aA + bB ⇌ cC + dD

K = [C]^c [D]^d / [A]^a [B]^b

重要性质：

  • K只与温度有关，与浓度、压强无关
  • 温度升高，若K增大，则正反应吸热；若K减小，则正反应放热

计算步骤（三段式）：

  1. 起始浓度
  2. 变化浓度
  3. 平衡浓度
  4. 代入K表达式

例题： N₂ + 3H₂ ⇌ 2NH₃，起始[N₂]=1 mol/L，[H₂]=3 mol/L，平衡[NH₃]=0.5 mol/L，求K。

解：

        N₂    +    3H₂    ⇌    2NH₃
起始    1          3            0
变化   -0.25       -0.75        +0.5
平衡    0.75        2.25         0.5

K = [NH₃]² / ([N₂][H₂]³) = (0.5)² / (0.75 × 2.25³) = ...

3.4 等效平衡 ​

定义：在相同条件下，同一可逆反应从不同起始状态开始，达到平衡时各物质百分含量相同。

等效条件：

  • 恒温恒容：换算成同一边，物质的量相等
  • 恒温恒压：换算成同一边，物质的量之比相等

应用：简化计算，判断不同起始状态是否等效。

3.5 平衡图像分析 ​

常见图像类型：

  1. 速率-时间图（v-t图）

    • 某时刻速率突变 → 浓度/压强改变
    • 某时刻速率渐变 → 温度改变
    • 正逆速率同倍数变化 → 加催化剂或压强改变（等体反应）

  2. 转化率-温度图

    • 升温转化率增大 → 正反应吸热
    • 升温转化率减小 → 正反应放热

  3. 先拐先平数值大

    • 图像先拐弯的 → 温度高或压强大或加催化剂

第四章：水溶液中的离子平衡 ​

4.1 弱电解质的电离平衡 ​

电离平衡：弱电解质在水溶液中部分电离，存在电离平衡。

电离方程式（可逆号）：

CH₃COOH ⇌ CH₃COO⁻ + H⁺
NH₃·H₂O ⇌ NH₄⁺ + OH⁻

电离度（α）：

α = 已电离的分子数 / 总分子数 × 100%

4.2 水的电离与pH ​

水的电离：

H₂O ⇌ H⁺ + OH⁻
Kw = [H⁺][OH⁻] = 1×10⁻¹⁴（25℃）

pH计算：

pH = -lg[H⁺]
pOH = -lg[OH⁻]
pH + pOH = 14（25℃）

溶液酸碱性：

溶液[H⁺] vs [OH⁻]pH范围
酸性[H⁺] > [OH⁻]< 7
中性[H⁺] = [OH⁻]= 7
碱性[H⁺] < [OH⁻]> 7

4.3 盐类水解 ​

定义：盐电离出的离子与水电离出的H⁺或OH⁻结合生成弱电解质的反应。

水解规律：

盐的类型例子水解情况溶液酸碱性
强酸强碱盐NaCl、KNO₃不水解中性
强酸弱碱盐NH₄ClNH₄⁺水解酸性
弱酸强碱盐CH₃COONaCH₃COO⁻水解碱性
弱酸弱碱盐CH₃COONH₄双水解看相对强弱

水解方程式：

NH₄⁺ + H₂O ⇌ NH₃·H₂O + H⁺（溶液显酸性）
CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻（溶液显碱性）

4.4 离子浓度大小比较 ​

解题工具：

  1. 电荷守恒：溶液中正负电荷总量相等
  2. 物料守恒：某元素的各种形态浓度之和等于初始浓度
  3. 质子守恒：水电离出的H⁺和OH⁻浓度相等

例子：CH₃COONa溶液中

  • 电荷守恒：[Na⁺] + [H⁺] = [CH₃COO⁻] + [OH⁻]
  • 物料守恒：[Na⁺] = [CH₃COO⁻] + [CH₃COOH]
  • 离子浓度：[Na⁺] > [CH₃COO⁻] > [OH⁻] > [H⁺]

第五章：电化学（高考必考） ​

5.1 原电池 ​

定义：将化学能转化为电能的装置。

形成条件：

  1. 两种活泼性不同的电极
  2. 电解质溶液
  3. 形成闭合回路
  4. 能自发进行氧化还原反应

工作原理：

  • 负极（活泼金属）：失电子，氧化反应
  • 正极（不活泼金属）：得电子，还原反应
  • 电子从负极经外电路流向正极

例子（Cu-Zn原电池）：

  • 负极（Zn）：Zn - 2e⁻ = Zn²⁺
  • 正极（Cu）：2H⁺ + 2e⁻ = H₂↑
  • 总反应：Zn + 2H⁺ = Zn²⁺ + H₂↑

5.2 电解池 ​

定义：将电能转化为化学能的装置。

工作原理：

  • 阳极：与电源正极相连，失电子，氧化反应
  • 阴极：与电源负极相连，得电子，还原反应

离子放电顺序：

  • 阳极（失电子）：活泼金属电极 > S²⁻ > I⁻ > Br⁻ > Cl⁻ > OH⁻ > 含氧酸根
  • 阴极（得电子）：Ag⁺ > Fe³⁺ > Cu²⁺ > H⁺ > Fe²⁺ > Zn²⁺ > Al³⁺ > Mg²⁺ > Na⁺

5.3 原电池 vs 电解池对比 ​

对比项原电池电解池
能量转化化学能→电能电能→化学能
反应类型自发进行外加电源强迫
负极/阳极负极氧化阳极氧化
正极/阴极正极还原阴极还原
电子流向负极→正极电源负极→阴极→阳极→电源正极

口诀：「阳氧阴还」（阳极氧化，阴极还原）

FAQ（常见错误与解答） ​

Q1：化学平衡图像题怎么分析？ ​

A：三步法：

  1. 看坐标轴（什么随什么变化）
  2. 看曲线趋势（增大/减小/先增后减）
  3. 用勒夏特列原理解释

注意：「先拐先平数值大」——先拐弯的曲线温度高或压强大。

Q2：等效平衡怎么判断？ ​

A：

  • 恒温恒容：换算成方程式同一边，物质的量相等则等效
  • 恒温恒压：换算成方程式同一边，物质的量之比相等则等效
  • 技巧：极端假设法（全部转化为反应物或生成物）

Q3：原电池和电解池电极判断总是错怎么办？ ​

A：记住口诀「阳氧阴还」：

  • 阳极（原电池负极）：氧化反应，失电子
  • 阴极（原电池正极）：还原反应，得电子

电极反应式书写步骤：

  1. 判断电极（阳极/阴极）
  2. 确定反应物（溶液中的离子或电极本身）
  3. 写出产物
  4. 配平（原子守恒+电荷守恒）

Q4：离子浓度大小比较题怎么入手？ ​

A：

  1. 先写电荷守恒式（必须会）
  2. 判断溶液酸碱性
  3. 考虑电离和水解的程度
  4. 记住常见结论（如CH₃COONa溶液中[Na⁺]>[CH₃COO⁻]>[OH⁻]>[H⁺]）

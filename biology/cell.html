二、概念定义 ​

2.1 细胞的分子组成 ​

大量元素：C、H、O、N、P、S、K、Ca、Mg（含量占生物体总重量万分之一以上） 微量元素：Fe、Mn、Zn、Cu、B、Mo（含量少，但必不可少）

最基本元素：C（碳）——生物大分子的骨架

重要化合物：

化合物元素组成功能
水H、O良好溶剂、参与反应、运输物质
无机盐多种组成复杂化合物、维持生命活动
糖类C、H、O主要能源物质
脂质C、H、O（有的含N、P）储能、膜成分、调节
蛋白质C、H、O、N（有的含S）生命活动的主要承担者
核酸C、H、O、N、P遗传信息的携带者

2.2 蛋白质 ⭐ ​

基本单位：氨基酸（约20种）

氨基酸结构通式：

    H
    |
H₂N-C-COOH
    |
    R（侧链基团）

氨基酸特点：至少含有一个氨基（-NH₂）和一个羧基（-COOH），且连在同一个碳原子上

蛋白质的形成：

  • 脱水缩合：一个氨基酸的羧基与另一个氨基酸的氨基连接，脱去一分子水
  • 肽键：-CO-NH-
  • 多肽链：多个氨基酸连接形成

相关计算：

  • 肽键数 = 氨基酸数 - 肽链数
  • 蛋白质分子量 = 氨基酸总分子量 - 脱去水分子总分子量

蛋白质结构层次：

  1. 一级结构：氨基酸的排列顺序
  2. 二级结构：肽链的局部折叠（如α-螺旋）
  3. 三级结构：整条肽链的空间结构
  4. 四级结构：多条肽链的组合

蛋白质功能多样性（结构决定功能）：

  • 结构蛋白（如肌肉蛋白、胶原蛋白）
  • 功能蛋白：酶（催化）、抗体（免疫）、血红蛋白（运输）、胰岛素（调节）

2.3 核酸 ​

分类：

  • DNA（脱氧核糖核酸）：主要存在于细胞核，双链双螺旋
  • RNA（核糖核酸）：主要存在于细胞质，单链

基本单位：核苷酸

DNARNA
五碳糖脱氧核糖核糖
含氮碱基A、T、G、CA、U、G、C
结构双螺旋单链
分布主要在细胞核主要在细胞质
功能储存遗传信息传递遗传信息、参与蛋白质合成

2.4 细胞膜 ⭐ ​

流动镶嵌模型（桑格和尼克森提出）：

  • 磷脂双分子层：基本支架，具有流动性
  • 蛋白质：镶嵌、贯穿、覆盖在磷脂双分子层上，承担运输、识别等功能
  • 糖类：与蛋白质或脂质结合形成糖蛋白或糖脂，参与细胞识别

结构特点：流动性（磷脂分子和大多数蛋白质分子可以运动）

功能特性：选择透过性（控制物质进出）

物质跨膜运输方式：

运输方式方向是否需要载体是否耗能示例
自由扩散高→低否否O₂、CO₂、水、甘油
协助扩散高→低是否葡萄糖进入红细胞
主动运输低→高是是小肠吸收葡萄糖、氨基酸
胞吞/胞吐--是大分子物质（蛋白质）

2.5 细胞器 ​

细胞器结构特点功能分布
线粒体双层膜，内膜折叠成嵴有氧呼吸主要场所，产生ATP动植物细胞
叶绿体双层膜，内有类囊体光合作用场所植物绿色细胞
核糖体无膜，由rRNA和蛋白质组成蛋白质合成场所所有细胞
内质网单层膜，网状蛋白质加工、脂质合成动植物细胞
高尔基体单层膜，扁平囊状蛋白质加工、分类、包装动植物细胞
溶酶体单层膜，含水解酶分解衰老细胞器、吞噬病原体动物细胞
液泡单层膜，含细胞液储存物质、维持细胞形态植物细胞
中心体无膜，由两个中心粒组成与细胞分裂有关动物细胞、低等植物

分泌蛋白的合成和分泌路径：

核糖体（合成肽链）→ 内质网（加工）→ 囊泡 → 高尔基体（再加工）→ 囊泡 → 细胞膜（分泌）

2.6 细胞代谢 ⭐ ​

2.6.1 酶 ​

本质：活细胞产生的具有催化作用的有机物，绝大多数是蛋白质，少数是RNA

特性：

  • 高效性：催化效率远高于无机催化剂
  • 专一性：一种酶只能催化一种或一类反应
  • 作用条件温和：需要适宜的温度和pH

影响酶活性的因素：

  • 温度：低温抑制活性，高温使酶失活（变性）
  • pH：过酸或过碱使酶失活

酶活性曲线：

  • 温度-pH曲线：倒U形，最适温度/pH时活性最高

2.6.2 ATP ​

结构：腺苷（腺嘌呤+核糖）+ 三个磷酸基团

功能：直接能源物质

ATP与ADP的相互转化：

ATP ⇌ ADP + Pi + 能量
  • 水解：释放能量，供生命活动
  • 合成：储存能量，来自光合作用或细胞呼吸

注意：这不是可逆反应（酶不同、场所不同、能量来源不同）

2.6.3 细胞呼吸 ⭐ ​

概念：有机物在细胞内经过一系列的氧化分解，生成CO₂或其他产物，释放能量并合成ATP的过程

有氧呼吸（主要场所：线粒体）：

阶段场所反应物产物能量变化
第一阶段细胞质基质葡萄糖丙酮酸 + [H]少量ATP
第二阶段线粒体基质丙酮酸 + H₂OCO₂ + [H]少量ATP
第三阶段线粒体内膜[H] + O₂H₂O大量ATP

总反应式：

C₆H₁₂O₆ + 6H₂O + 6O₂ → 6CO₂ + 12H₂O + 能量

无氧呼吸（场所：细胞质基质）：

类型生物产物反应式
酒精发酵大多数植物、酵母菌酒精 + CO₂C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂ + 少量能量
乳酸发酵动物、乳酸菌、少数植物乳酸C₆H₁₂O₆ → 2C₃H₆O₃ + 少量能量

有氧呼吸vs无氧呼吸比较：

比较项目有氧呼吸无氧呼吸
条件需O₂不需O₂
场所细胞质基质、线粒体细胞质基质
产物CO₂、H₂O酒精+CO₂或乳酸
能量释放大量少量
是否彻底分解是否

2.6.4 光合作用 ⭐ ​

概念：绿色植物通过叶绿体，利用光能，把CO₂和H₂O转化成储存能量的有机物，并释放O₂的过程

光反应（场所：类囊体薄膜）：

水的光解：H₂O → [H] + O₂（需要光）
ATP合成：ADP + Pi + 光能 → ATP
  • 条件：光、色素、酶
  • 产物：[H]（NADPH）、ATP、O₂

暗反应（场所：叶绿体基质）：

CO₂固定：CO₂ + C₅ → 2C₃
C₃还原：2C₃ + [H] + ATP → (CH₂O) + C₅
  • 条件：酶、[H]、ATP
  • 产物：有机物(CH₂O)、C₅（再生）

光合作用总反应式：

CO₂ + H₂O → (CH₂O) + O₂（条件：光能、叶绿体）

光合作用与细胞呼吸的关系：

  • 光合作用：合成有机物，储存能量
  • 细胞呼吸：分解有机物，释放能量
  • 二者相互依存，共同维持生命活动

2.7 细胞的生命历程 ​

2.7.1 有丝分裂 ⭐ ​

过程（以植物细胞为例）：

时期主要特征
间期DNA复制，有关蛋白质合成，细胞适度生长
前期核膜、核仁消失，染色体、纺锤体出现
中期染色体着丝点排列在赤道板上，形态最清晰
后期着丝点分裂，姐妹染色单体分开，染色体数目加倍
末期核膜、核仁重现，染色体、纺锤体消失，细胞板形成

特点：亲代细胞染色体复制后，平均分配到两个子细胞中，子细胞与亲代细胞染色体数目相同

意义：保持亲子代细胞遗传信息的一致性

2.7.2 细胞分化 ​

概念：在个体发育中，由一个或一种细胞增殖产生的后代，在形态、结构和功能上发生稳定性差异的过程

实质：基因的选择性表达（不同细胞表达的基因不同）

特点：

  • 普遍性：多细胞生物普遍存在
  • 持久性：贯穿生命全过程
  • 稳定性：一般不可逆转
  • 不可逆性：分化后的细胞一般不能再分化成其他细胞

2.7.3 细胞衰老、凋亡、癌变 ​

细胞衰老特征：

  • 水分减少，细胞萎缩
  • 酶活性降低，代谢减慢
  • 色素积累
  • 细胞膜通透性改变
  • 细胞核体积增大，核膜内折

细胞凋亡：

  • 概念：由基因决定的细胞自动结束生命的过程（细胞编程性死亡）
  • 意义：完成正常发育，维持内部环境稳定，抵御外界干扰

细胞癌变：

  • 特征：无限增殖、形态结构改变、易分散转移
  • 原因：原癌基因和抑癌基因发生突变

三、典型例题 ​

例题1：蛋白质相关计算 ​

题目：某蛋白质由n条肽链组成，共有m个氨基酸，求：

  • (1) 肽键数
  • (2) 脱去水分子数
  • (3) 若氨基酸平均分子量为a，求蛋白质分子量

解答：

(1) 肽键数 = 氨基酸数 - 肽链数 = m - n

(2) 脱去水分子数 = 肽键数 = m - n

(3) 蛋白质分子量 = 氨基酸总分子量 - 脱去水分子总分子量
                  = m·a - (m-n)·18

例题2：物质跨膜运输判断 ​

题目：下列物质通过细胞膜的方式分别是？

  • (1) O₂进入肺泡细胞
  • (2) 葡萄糖进入红细胞
  • (3) K⁺进入植物根细胞
  • (4) 蛋白质分泌到细胞外

解答：

(1) 自由扩散（O₂是小分子，顺浓度梯度）
(2) 协助扩散（需要载体，顺浓度梯度，不耗能）
(3) 主动运输（逆浓度梯度，需要载体和能量）
(4) 胞吐（大分子，通过囊泡分泌）

例题3：光合作用和呼吸作用计算 ​

题目：某植物在黑暗条件下释放CO₂ 6mg/h，在光照条件下（光合速率大于呼吸速率）吸收CO₂ 18mg/h。求：

  • (1) 呼吸速率
  • (2) 总光合速率
  • (3) 净光合速率

解答：

(1) 呼吸速率 = 6 mg CO₂/h

(2) 总光合速率 = 净光合速率 + 呼吸速率
               = 18 + 6 
               = 24 mg CO₂/h

(3) 净光合速率 = 18 mg CO₂/h（题目给出）

关系：总光合速率 = 净光合速率 + 呼吸速率

五、FAQ常见问题 ​

FAQ 1：蛋白质计算总是错 ​

常见错误：

  • 忘记减去肽链数
  • 忘记水的分子量是18
  • 环状肽的特殊计算

记忆方法：

  • "手拉手"：n个人手拉手，形成n-1个连接
  • 环状肽：n个人围成圈，形成n个连接

FAQ 2：细胞器的功能总记混 ​

分类记忆：

  • 与能量有关：线粒体（细胞呼吸）、叶绿体（光合作用）
  • 与蛋白质有关：核糖体（合成）、内质网（加工）、高尔基体（再加工）
  • 其他：溶酶体（分解）、液泡（储存）、中心体（分裂）

FAQ 3：光合作用和呼吸作用过程理不清 ​

建议：

  1. 画流程图，贴在显眼位置
  2. 分阶段记忆，不要一次记全部
  3. 理解"光反应为暗反应提供[H]和ATP"
  4. 定期复述流程

关键节点：

  • 水光解 → [H] + O₂
  • CO₂固定 → C₃
  • C₃还原 → 需要[H]和ATP

FAQ 4：有丝分裂各期特点记不住 ​

口诀：

  • "前期膜仁消失现两体"（核膜核仁消失，染色体纺锤体出现）
  • "中期形定数晰赤道齐"（染色体形态固定，数目清晰，排列在赤道板）
  • "后期点裂数加均两极"（着丝点分裂，染色体数目加倍，移向两极）
  • "末期两消两现重开始"（染色体纺锤体消失，核膜核仁重现）

二、概念定义 ​

2.1 基因工程 ⭐ ​

概念：按照人们的意愿，把一种生物的某种基因提取出来，加以修饰改造，然后放到另一种生物的细胞里，定向地改造生物的遗传性状

别名：基因拼接技术、DNA重组技术

2.1.1 基因工程的基本工具 ​

工具1：限制性核酸内切酶（限制酶）

  • 来源：主要从原核生物中分离纯化
  • 功能：识别双链DNA分子的某种特定核苷酸序列，并使每一条链中特定部位的两个核苷酸之间的磷酸二酯键断开
  • 结果：产生黏性末端或平末端
  • 特点：一种限制酶只能识别一种特定的核苷酸序列

工具2：DNA连接酶

  • 功能：将双链DNA片段"缝合"起来，恢复被限制酶切开的磷酸二酯键
  • 类型：E·coli DNA连接酶、T₄ DNA连接酶

工具3：载体

  • 功能：将外源基因送入受体细胞
  • 常用载体：质粒（最常用）、λ噬菌体的衍生物、动植物病毒
  • 质粒特点：
    • 独立于细菌染色体之外的双链环状DNA分子
    • 具有自我复制能力
    • 具有一个至多个限制酶切点
    • 具有标记基因（供重组DNA的鉴定和选择）

2.1.2 基因工程的基本操作程序⭐ ​

四步曲：

第一步：目的基因的获取

  • 方法：
    • 从基因文库中获取
    • PCR技术扩增
    • 人工合成（已知序列的小基因）

第二步：基因表达载体的构建（核心步骤）

  • 目的：使目的基因在受体细胞中稳定存在，并且可以遗传给下一代，同时使目的基因能够表达和发挥作用
  • 组成：
    • 目的基因
    • 启动子（RNA聚合酶识别和结合的部位，驱动基因转录）
    • 终止子
    • 标记基因（鉴别受体细胞是否含有目的基因）

第三步：将目的基因导入受体细胞

受体细胞导入方法
植物细胞农杆菌转化法（最常用）、基因枪法、花粉管通道法
动物细胞显微注射法
微生物细胞Ca²⁺处理法（感受态细胞）

第四步：目的基因的检测与鉴定

检测水平方法
分子水平DNA分子杂交（检测目的基因）、分子杂交（检测mRNA）、抗原-抗体杂交（检测蛋白质）
个体水平抗虫/抗病接种实验

2.1.3 PCR技术 ​

概念：多聚酶链式反应，体外快速扩增DNA片段的技术

原理：DNA双链复制

条件：

  • 模板DNA
  • 两种引物
  • 四种脱氧核苷酸
  • Taq DNA聚合酶（热稳定DNA聚合酶）
  • 缓冲液、Mg²⁺等

过程（循环进行）：

  1. 变性：90-95℃，DNA双链解开
  2. 复性：55-60℃，引物与模板结合
  3. 延伸：70-75℃，Taq酶合成新链

特点：

  • 指数扩增：n次循环后，DNA数量增加2ⁿ倍
  • 快速：几小时可扩增百万倍

2.1.4 基因工程的应用 ​

植物基因工程：

  • 抗虫转基因植物（Bt毒蛋白基因）
  • 抗病转基因植物
  • 抗逆转基因植物
  • 改良植物品质

动物基因工程：

  • 提高生长速度
  • 改善畜产品品质
  • 生产药物（乳腺生物反应器）
  • 作器官移植的供体

基因工程药物：

  • 胰岛素、干扰素、白细胞介素等

基因治疗：

  • 概念：把正常基因导入病人体内，使该基因的表达产物发挥功能，从而达到治疗疾病的目的
  • 类型：体外基因治疗、体内基因治疗

2.1.5 蛋白质工程 ​

概念：以蛋白质分子的结构规律及其与生物功能的关系作为基础，通过基因修饰或基因合成，对现有蛋白质进行改造，或制造一种新的蛋白质

与基因工程的关系：

  • 基因工程：生产自然界已有的蛋白质
  • 蛋白质工程：生产自然界不存在的蛋白质（第二代基因工程）

流程：

预期蛋白质功能 → 设计蛋白质结构 → 推测氨基酸序列 → 找到对应的脱氧核苷酸序列（基因）→ 基因工程生产

2.2 细胞工程 ​

概念：应用细胞生物学和分子生物学的原理和方法，通过细胞水平或细胞器水平上的操作，按照人的意愿来改变细胞内的遗传物质或获得细胞产品

2.2.1 植物细胞工程 ​

技术1：植物组织培养

原理：植物细胞的全能性（已分化的细胞仍具有发育成完整个体的潜能）

过程：

外植体（离体的组织、器官或细胞）→ 脱分化 → 愈伤组织 → 再分化 → 胚状体/丛芽 → 试管苗 → 完整植株

条件：

  • 无菌环境
  • 适宜的培养基（含植物激素：生长素和细胞分裂素）
  • 适宜的温度、光照

应用：

  • 快速繁殖
  • 培育脱毒苗
  • 生产人工种子
  • 生产细胞产物

技术2：植物体细胞杂交

过程：

植物细胞A（去壁）→ 原生质体A \
                    → 融合 → 杂种细胞 → 愈伤组织 → 杂种植株
植物细胞B（去壁）→ 原生质体B /

意义：克服远缘杂交不亲和的障碍

实例：白菜-甘蓝、番茄-马铃薯

2.2.2 动物细胞工程 ​

技术1：动物细胞培养

原理：细胞增殖

条件：

  • 无菌、无毒环境
  • 营养（糖、氨基酸、促生长因子、无机盐、微量元素、血清）
  • 适宜温度和pH
  • 气体环境（O₂和CO₂）

过程：原代培养 → 传代培养

应用：生产病毒疫苗、干扰素、单克隆抗体等

技术2：动物细胞核移植（克隆技术）

概念：将动物的一个细胞的细胞核，移入一个已经去掉细胞核的卵母细胞中，使其重组并发育成一个新的胚胎，最终发育为动物个体

分类：

  • 胚胎细胞核移植（较易）
  • 体细胞核移植（较难）

实例：多莉羊

应用：

  • 加速家畜遗传改良
  • 保护濒危物种
  • 生产医用蛋白
  • 作为器官移植的供体

技术3：动物细胞融合

诱导因素：PEG（聚乙二醇）、灭活的病毒、电激

最重要应用：单克隆抗体的制备

单克隆抗体制备过程：

注射抗原 → 小鼠产生免疫反应 → 取出脾脏B淋巴细胞
                                    ↓
骨髓瘤细胞（无限增殖）────────→ 细胞融合 → 杂交瘤细胞
                                    ↓
                筛选（选择性培养基）→ 克隆化培养、抗体检测
                                    ↓
                            产生专一抗体的杂交瘤细胞
                                    ↓
                        体外培养或小鼠腹腔内增殖 → 单克隆抗体

单克隆抗体优点：

  • 特异性强
  • 灵敏度高
  • 可大量制备

应用：诊断试剂、治疗疾病、运载药物（生物导弹）

2.2.3 胚胎工程 ​

概念：对动物早期胚胎或配子所进行的多种显微操作和处理技术

主要技术：

  • 体外受精
  • 胚胎移植
  • 胚胎分割
  • 胚胎干细胞培养

胚胎移植：

  • 概念：将雌性动物的早期胚胎，或者通过体外受精及其他方式得到的胚胎，移植到同种的、生理状态相同的其他雌性动物体内，使之继续发育为新个体的技术
  • 地位：胚胎工程的最后一道"工序"
  • 意义：充分发挥雌性优良个体的繁殖潜力

胚胎分割：

  • 概念：采用机械方法将早期胚胎切割成2等份、4等份或8等份等，经移植获得同卵双胎或多胎的技术
  • 特点：后代遗传物质相同（无性繁殖）
  • 要求：分割囊胚阶段的胚胎时，要将内细胞团均等分割

2.3 发酵工程 ​

概念：利用微生物的特定功能，通过现代工程技术，规模化生产对人类有用的产品

基本环节：

  1. 菌种的选育：诱变育种、基因工程育种
  2. 培养基的配制：根据微生物营养需求配制
  3. 灭菌：防止杂菌污染
  4. 接种：将菌种接入培养基
  5. 发酵过程：控制温度、pH、溶氧量等
  6. 产品的分离提纯

影响发酵过程的因素：

  • 温度：影响酶的活性
  • pH：影响酶的活性和细胞膜通透性
  • 溶氧量：好氧菌需要充足氧气，厌氧菌需要无氧环境
  • 营养物质：碳源、氮源、无机盐等

应用：

  • 食品工业：酿酒、制醋、生产酸奶、味精等
  • 医药工业：生产抗生素、维生素、激素等
  • 能源开发：生产乙醇、沼气等

三、生物技术的安全性和伦理问题 ​

3.1 转基因生物的安全性 ​

争论焦点：

  • 食物安全：毒性蛋白、过敏原、营养成分改变
  • 生物安全：对生物多样性的影响
  • 环境安全：对生态系统稳定性的影响

3.2 生物武器 ​

特点：致病力强、传染性强、难以防治、成本低

禁止：1998年《禁止生物武器公约》

3.3 克隆技术的伦理问题 ​

  • 克隆人违反人类伦理道德
  • 冲击现有婚姻、家庭、两性关系
  • 心理和社会地位不确定

3.4 试管婴儿与设计试管婴儿 ​

试管婴儿：解决不孕不育问题 设计试管婴儿：用于救治患者（造血干细胞移植），但引发伦理争议

四、典型例题 ​

例题1：基因工程操作步骤 ​

题目：简述基因工程的基本操作程序。

解答：

第一步：获取目的基因
第二步：构建基因表达载体（核心步骤）
第三步：将目的基因导入受体细胞
第四步：目的基因的检测与鉴定

例题2：PCR技术 ​

题目：PCR技术为什么需要引物？为什么使用Taq DNA聚合酶？

解答：

需要引物：DNA聚合酶不能从头开始合成DNA，只能在3'端延伸，
所以需要引物提供3'-OH末端。

使用Taq酶：PCR过程中需要反复加热到90-95℃使DNA变性，
普通DNA聚合酶会失活，而Taq DNA聚合酶具有热稳定性，
在高温下仍保持活性。

例题3：单克隆抗体制备 ​

题目：为什么要将B淋巴细胞与骨髓瘤细胞融合？

解答：

B淋巴细胞能产生特异性抗体，但在体外不能无限增殖；
骨髓瘤细胞能无限增殖，但不能产生抗体。

融合形成的杂交瘤细胞既能产生特异性抗体，又能无限增殖，
可以在体外大规模培养，生产单克隆抗体。

五、配套练习 ​

基础练习 ​

练习1：基因工程的三种基本工具是什么？它们各自的功能是什么？

练习2：植物组织培养的原理是什么？为什么要强调无菌操作？

练习3：单克隆抗体有什么优点？主要应用在哪些方面？

参考答案：

  • 练习1：限制酶（切割DNA）、DNA连接酶（连接DNA片段）、载体（运送目的基因）
  • 练习2：植物细胞的全能性；防止微生物污染，与植物细胞竞争营养
  • 练习3：特异性强、灵敏度高、可大量制备；诊断试剂、治疗疾病、运载药物

六、FAQ常见问题 ​

FAQ 1：限制酶和DNA连接酶的作用部位？ ​

答：都是磷酸二酯键。限制酶切断磷酸二酯键，DNA连接酶连接磷酸二酯键。

FAQ 2：基因表达载体的构建为什么是核心步骤？ ​

答：因为只有构建好表达载体，目的基因才能在受体细胞中稳定存在、遗传给后代并正确表达。

FAQ 3：动物细胞培养为什么要加血清？ ​

答：血清中含有多种未知的促生长因子、激素等，目前还不能完全用人工合成培养基替代。

FAQ 4：胚胎分割为什么属于无性繁殖？ ​

答：因为分割后的胚胎来自同一个受精卵，遗传物质相同，没有经过两性生殖细胞的结合。

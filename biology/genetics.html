二、概念定义 ​

2.1 减数分裂 ⭐ ​

概念：进行有性生殖的生物，在产生成熟生殖细胞时进行的染色体数目减半的细胞分裂

过程（以精子形成为例）：

减数第一次分裂：

  • 前期Ⅰ：同源染色体联会，形成四分体，可能发生交叉互换
  • 中期Ⅰ：同源染色体排列在赤道板两侧
  • 后期Ⅰ：同源染色体分离，非同源染色体自由组合
  • 末期Ⅰ：细胞分裂，染色体数目减半

减数第二次分裂（类似有丝分裂）：

  • 前期Ⅱ：染色体散乱分布
  • 中期Ⅱ：染色体着丝点排列在赤道板上
  • 后期Ⅱ：着丝点分裂，姐妹染色单体分开
  • 末期Ⅱ：细胞分裂，形成4个精子细胞

减数分裂vs有丝分裂：

比较项目减数分裂有丝分裂
发生部位生殖腺体细胞
分裂次数2次1次
同源染色体配对有无
染色体数目变化减半（2n→n）不变（2n→2n）
子细胞数目4个2个
子细胞类型生殖细胞体细胞
意义产生配子，遗传变异生长、修复

2.2 DNA的结构 ⭐ ​

DNA双螺旋结构（沃森和克里克提出）：

结构特点：

  1. 反向平行：两条脱氧核苷酸链反向平行
  2. 外侧：磷酸和脱氧核糖交替连接，构成基本骨架
  3. 内侧：碱基通过氢键连接，形成碱基对
  4. 碱基互补配对原则：A=T（2个氢键），G≡C（3个氢键）

DNA分子特性：

  • 多样性：碱基排列顺序千变万化
  • 特异性：每个DNA分子有特定的碱基排列顺序
  • 稳定性：磷酸和脱氧核糖交替连接的稳定骨架

2.3 DNA的复制 ⭐ ​

概念：以亲代DNA为模板合成子代DNA的过程

时间：有丝分裂间期和减数第一次分裂前的间期

条件：

  • 模板：DNA的两条链
  • 原料：4种游离的脱氧核苷酸
  • 能量：ATP
  • 酶：解旋酶、DNA聚合酶等

过程：

  1. 解旋：解旋酶作用下，双链打开
  2. 合成子链：DNA聚合酶作用下，按碱基互补配对原则合成新链
  3. 形成子代DNA：每条新链与对应的模板链形成双螺旋

特点：

  • 半保留复制：新合成的每个DNA分子中，都保留了原来DNA分子中的一条链
  • 边解旋边复制

意义：保持遗传信息的连续性

2.4 基因的表达 ⭐ ​

2.4.1 转录 ​

概念：以DNA的一条链为模板合成RNA的过程

场所：主要在细胞核

过程：

DNA（基因的一条链）→ mRNA

碱基配对：A-U、T-A、G-C、C-G

结果：遗传信息从DNA传递到mRNA

2.4.2 翻译 ​

概念：以mRNA为模板合成蛋白质的过程

场所：核糖体

过程：

  1. 起始：mRNA与核糖体结合
  2. 延伸：tRNA携带氨基酸，按mRNA上的密码子顺序，在核糖体上连接成肽链
  3. 终止：遇到终止密码子，肽链释放

关键概念：

  • 密码子：mRNA上决定一个氨基酸的3个相邻碱基（64种，其中3种终止密码子）
  • 反密码子：tRNA上与mRNA密码子互补配对的3个碱基
  • 氨基酸：20种

数量关系：

  • DNA碱基数 : mRNA碱基数 : 氨基酸数 = 6 : 3 : 1

2.4.3 中心法则 ​

DNA →（转录）→ RNA →（翻译）→ 蛋白质
  ↓（复制）
DNA

补充（某些病毒）：

  • RNA自我复制
  • 逆转录：RNA→DNA（逆转录病毒，如HIV）

2.5 遗传的基本规律 ⭐ ​

2.5.1 分离定律（孟德尔第一定律） ​

内容：在生物的体细胞中，控制同一性状的遗传因子成对存在，不相融合；在形成配子时，成对的遗传因子发生分离，分离后的遗传因子分别进入不同的配子中，随配子遗传给后代

实质：等位基因随同源染色体的分离而分离

验证：测交实验（F₁与隐性纯合子杂交）

应用：

  • 预测后代性状分离比（3:1、1:1）
  • 系谱分析

2.5.2 自由组合定律（孟德尔第二定律） ​

内容：控制不同性状的遗传因子的分离和组合是互不干扰的；在形成配子时，决定同一性状的成对的遗传因子彼此分离，决定不同性状的遗传因子自由组合

实质：非同源染色体上的非等位基因自由组合

验证：测交实验（双杂合子与双隐性纯合子杂交，后代比例1:1:1:1）

应用：

  • 预测后代性状分离比（9:3:3:1及其变式）
  • 杂交育种

2.5.3 遗传概率计算方法 ​

方法一：棋盘法

  • 列出雌雄配子
  • 填写后代基因型和表现型
  • 统计比例

方法二：分支法

  • 逐对分析每对性状
  • 用乘法原理计算组合概率

示例：AaBb × Aabb

  • Aa × Aa → 3/4 A_ : 1/4 aa
  • Bb × bb → 1/2 Bb : 1/2 bb
  • 组合：A_Bb = 3/4 × 1/2 = 3/8

2.6 伴性遗传 ​

概念：基因位于性染色体上，遗传上总是和性别相关联

类型：

类型实例特点
X染色体隐性遗传红绿色盲、血友病男性患者多于女性，交叉遗传
X染色体显性遗传抗维生素D佝偻病女性患者多于男性，世代连续
Y染色体遗传外耳道多毛症只有男性患病，父传子

系谱图分析步骤：

  1. 判断显隐性："无中生有"为隐性，"有中生无"为显性
  2. 判断染色体位置：是否与性别相关联
  3. 写出基因型：根据表现型推断
  4. 计算概率：确定范围后计算

2.7 生物的变异 ​

2.7.1 基因突变 ​

概念：DNA分子中发生碱基对的替换、增添和缺失，而引起的基因结构的改变

特点：

  • 普遍性
  • 随机性
  • 低频性
  • 不定向性
  • 多害少利性

意义：产生新基因，是生物变异的根本来源

2.7.2 基因重组 ​

类型：

  1. 减数第一次分裂后期：非同源染色体上的非等位基因自由组合
  2. 减数第一次分裂前期：同源染色体的非姐妹染色单体交叉互换
  3. 基因工程：体外DNA重组

意义：产生多样化的基因组合，是生物变异的重要来源

2.7.3 染色体变异 ​

类型：

  • 结构变异：缺失、重复、倒位、易位
  • 数目变异：个别染色体增减、染色体组成倍增减（多倍体、单倍体）

2.8 现代生物进化理论 ​

核心内容：

  1. 种群是生物进化的基本单位
  2. 突变和基因重组产生进化的原材料
  3. 自然选择决定生物进化的方向
  4. 隔离导致新物种的形成

基因频率和基因型频率：

  • 基因频率 = 该基因个数 / 该基因及其等位基因总个数
  • 基因型频率 = 该基因型个体数 / 总个体数

哈代-温伯格定律（理想条件下）：

  • (p+q)² = p² + 2pq + q² = 1
  • p：A基因频率，q：a基因频率
  • p²：AA基因型频率，2pq：Aa基因型频率，q²：aa基因型频率

三、典型例题 ​

例题1：遗传概率计算 ​

题目：豌豆的黄色（Y）对绿色（y）为显性，圆粒（R）对皱粒（r）为显性。现有黄色圆粒（YyRr）与绿色皱粒（yyrr）杂交，求后代的基因型和表现型比例。

解答：

YyRr产生的配子：YR、Yr、yR、yr（各1/4）
yyrr产生的配子：yr（1）

测交后代：
YyRr : Yyrr : yyRr : yyrr = 1:1:1:1

表现型：
黄色圆粒 : 黄色皱粒 : 绿色圆粒 : 绿色皱粒 = 1:1:1:1

例题2：系谱图分析 ​

题目：某遗传病系谱图显示：

  • Ⅰ代：双亲正常，生育一患病儿子
  • Ⅱ代：患病男性与正常女性婚配，生育一患病儿子

问：该病的遗传方式是什么？

解答：

1. 判断显隐性："无中生有"为隐性
2. 判断染色体位置：男性患者多于女性，交叉遗传
3. 结论：X染色体隐性遗传

例题3：DNA复制计算 ​

题目：某DNA分子含有1000个碱基对，其中腺嘌呤（A）占30%。该DNA分子连续复制3次，问：

  • (1) 需要多少个游离的鸟嘌呤（G）？
  • (2) 第3次复制需要多少个G？

解答：

(1) A = T = 30%，则 G = C = 20%
    G的数量 = 1000 × 2 × 20% = 400个
    
    复制3次，产生8个DNA分子
    需要游离G = 400 × (8-1) = 2800个

(2) 第3次复制，从4个DNA分子复制到8个
    需要游离G = 400 × 4 = 1600个

四、配套练习 ​

基础练习 ​

练习1：Aa自交，F₁中纯合子占多少？

练习2：人类红绿色盲是X染色体隐性遗传病。一正常女性（其父为色盲）与一正常男性婚配，生育色盲男孩的概率是多少？

练习3：DNA复制时，一条链上A:T:G:C = 1:2:3:4，求另一条链和整个DNA分子中四种碱基的比例。

参考答案：

  • 练习1：1/2
  • 练习2：1/8
  • 练习3：另一条链T:A:C:G = 1:2:3:4；整个DNA中A:T:G:C = 3:3:7:7

五、FAQ常见问题 ​

FAQ 1：分离比总记混 ​

记忆：

  • 杂合子自交：3:1
  • 测交：1:1
  • 双杂合子自交：9:3:3:1
  • 双杂合子测交：1:1:1:1

FAQ 2：伴性遗传判断不清 ​

方法：

  • 看男女患者比例
  • 男性患者多→X隐性
  • 女性患者多→X显性
  • 只有男性→Y遗传

FAQ 3：DNA复制计算总错 ​

公式：

  • 复制n次，需要游离某种脱氧核苷酸 = m × (2ⁿ - 1)
  • 第n次复制需要 = m × 2ⁿ⁻¹ （m为DNA分子中该核苷酸数）

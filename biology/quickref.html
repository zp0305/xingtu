生物知识速查卡 ​

考前30分钟快速浏览——核心概念与结论

一、分子与细胞 ​

细胞的分子组成 ​

  • 最基本元素：C
  • 主要能源物质：糖类
  • 储能物质：脂肪
  • 生命活动的主要承担者：蛋白质
  • 遗传信息的携带者：核酸

蛋白质 ​

  • 基本单位：氨基酸（20种）
  • 肽键数 = 氨基酸数 - 肽链数
  • 结构层次：氨基酸→多肽链→蛋白质
  • 功能：结构蛋白、酶、抗体、载体、激素

核酸 ​

  • DNA：双螺旋，A-T、G-C配对
  • RNA：单链，A-U、G-C配对

细胞膜 ​

  • 结构：流动镶嵌模型
  • 结构特点：流动性
  • 功能特性：选择透过性
  • 运输方式：自由扩散、协助扩散、主动运输、胞吞胞吐

细胞器 ​

  • 线粒体：有氧呼吸主要场所
  • 叶绿体：光合作用场所
  • 核糖体：蛋白质合成
  • 内质网：蛋白质加工、脂质合成
  • 高尔基体：蛋白质加工、分类、包装
  • 溶酶体：含水解酶，分解衰老细胞器

细胞代谢 ​

酶：

  • 本质：大多数是蛋白质，少数是RNA
  • 特性：高效性、专一性、作用条件温和
  • 影响因素：温度、pH

ATP：

  • 结构：腺苷 + 三个磷酸基团
  • 功能：直接能源物质
  • ATP ⇌ ADP + Pi + 能量

有氧呼吸：

  • 第一阶段：细胞质基质，葡萄糖→丙酮酸 + [H]，少量ATP
  • 第二阶段：线粒体基质，丙酮酸 + H₂O → CO₂ + [H]，少量ATP
  • 第三阶段：线粒体内膜，[H] + O₂ → H₂O，大量ATP

无氧呼吸：

  • 酒精发酵：C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂ + 少量能量
  • 乳酸发酵：C₆H₁₂O₆ → 2C₃H₆O₃ + 少量能量

光合作用：

  • 光反应（类囊体薄膜）：H₂O → [H] + O₂，ADP + Pi → ATP
  • 暗反应（叶绿体基质）：CO₂ + C₅ → 2C₃，2C₃ + [H] + ATP → (CH₂O) + C₅

关系：总光合速率 = 净光合速率 + 呼吸速率

细胞生命历程 ​

  • 有丝分裂：间期→前期→中期→后期→末期
  • 分化实质：基因的选择性表达
  • 细胞全能性：已分化细胞仍具有发育成完整个体的潜能
  • 衰老特征：水分减少、酶活性降低、色素积累
  • 凋亡：基因决定的细胞编程性死亡
  • 癌变：原癌基因和抑癌基因突变

二、遗传与进化 ​

减数分裂 ​

  • 减Ⅰ：同源染色体分离，染色体数目减半
  • 减Ⅱ：着丝点分裂，姐妹染色单体分开
  • 结果：4个生殖细胞，染色体数目减半

DNA结构 ​

  • 双螺旋：反向平行
  • 外侧：磷酸-脱氧核糖骨架
  • 内侧：碱基对（A=T，G≡C）

DNA复制 ​

  • 时间：间期
  • 方式：半保留复制
  • 条件：模板、原料、能量、酶

基因表达 ​

  • 转录：DNA → mRNA（A-U、T-A、G-C、C-G）
  • 翻译：mRNA → 蛋白质（密码子、反密码子）
  • 数量关系：DNA碱基数 : mRNA碱基数 : 氨基酸数 = 6:3:1

遗传规律 ​

  • 分离定律：等位基因随同源染色体分离而分离
  • 自由组合定律：非同源染色体上的非等位基因自由组合
  • 分离比：3:1（自交）、1:1（测交）、9:3:3:1（双杂合自交）

伴性遗传 ​

  • X隐性：男患者多于女患者，交叉遗传（如色盲、血友病）
  • X显性：女患者多于男患者（如抗维生素D佝偻病）

变异 ​

  • 基因突变：DNA分子中碱基对的替换、增添、缺失
  • 基因重组：减Ⅰ后期自由组合、前期交叉互换
  • 染色体变异：结构变异、数目变异

进化 ​

  • 基本单位：种群
  • 原材料：突变和基因重组
  • 方向决定：自然选择
  • 新物种形成：隔离

三、稳态与调节 ​

神经调节 ​

  • 结构基础：反射弧（感受器→传入神经→神经中枢→传出神经→效应器）
  • 兴奋传导：静息电位（外正内负，K⁺外流）→动作电位（外负内正，Na⁺内流）
  • 突触传递：电信号→化学信号（神经递质）→电信号，单向传递

体液调节 ​

  • 激素特点：微量高效、体液运输、作用于靶器官靶细胞
  • 甲状腺激素：促进新陈代谢和生长发育
  • 胰岛素：降低血糖
  • 胰高血糖素：升高血糖
  • 分级调节：下丘脑→垂体→甲状腺

血糖调节 ​

  • 升高→胰岛素分泌→降血糖
  • 降低→胰高血糖素分泌→升血糖
  • 关系：拮抗作用、反馈调节

免疫调节 ​

  • 三道防线：皮肤黏膜（第一道）、体液中的杀菌物质和吞噬细胞（第二道）、免疫器官和免疫细胞（第三道）
  • 特异性免疫：
    • 体液免疫：B细胞→浆细胞→抗体，对付细胞外病原体
    • 细胞免疫：T细胞→效应T细胞，对付细胞内病原体
  • 记忆细胞：快速二次免疫

植物激素 ​

  • 生长素：两重性（低促高抑），极性运输
  • 赤霉素：促进细胞伸长
  • 细胞分裂素：促进细胞分裂
  • 脱落酸：抑制生长，促进脱落
  • 乙烯：促进果实成熟

四、生态与环境 ​

种群 ​

  • 数量特征：种群密度（最基本）、出生率、死亡率、迁入率、迁出率、年龄组成、性别比例
  • J型曲线：理想条件，Nt = N₀λᵗ
  • S型曲线：有限环境，K值（环境容纳量），K/2时增长速率最快

群落 ​

  • 种间关系：互利共生、捕食、竞争、寄生
  • 空间结构：垂直结构（分层）、水平结构（镶嵌）
  • 演替：初生演替、次生演替

生态系统 ​

  • 组成：非生物物质和能量、生产者、消费者、分解者
  • 能量流动：单向流动、逐级递减（10%-20%）
  • 物质循环：碳循环（CO₂形式）、氮循环
  • 稳定性：抵抗力稳定性、恢复力稳定性（一般负相关）

生物多样性 ​

  • 三个层次：基因多样性、物种多样性、生态系统多样性
  • 保护措施：就地保护（最有效）、易地保护

五、生物技术 ​

基因工程 ​

  • 工具：限制酶（切割）、DNA连接酶（连接）、载体（运输）
  • 四步骤：获取目的基因→构建表达载体→导入受体细胞→检测与鉴定
  • PCR：变性→复性→延伸，指数扩增
  • 应用：转基因作物、基因药物、基因治疗

植物细胞工程 ​

  • 组织培养原理：植物细胞全能性
  • 过程：外植体→愈伤组织→胚状体→植株
  • 体细胞杂交：克服远缘杂交不亲和

动物细胞工程 ​

  • 细胞培养：原代培养、传代培养
  • 核移植：克隆
  • 细胞融合：单克隆抗体制备

单克隆抗体 ​

  • 优点：特异性强、灵敏度高、可大量制备
  • 应用：诊断试剂、治疗疾病、生物导弹

胚胎工程 ​

  • 胚胎移植：最后一道工序
  • 胚胎分割：无性繁殖，内细胞团均等分割

发酵工程 ​

  • 影响因素：温度、pH、溶氧量
  • 应用：食品（酒、醋、酸奶）、药物（抗生素、维生素）

六、高频易错点 ​

  1. 酶可以重复利用，在反应前后性质不变
  2. 光合作用产生的ATP仅供暗反应使用，不用于其他生命活动
  3. 有氧呼吸产生的CO₂中的O来自葡萄糖和水，不是来自O₂
  4. DNA复制是半保留，不是全保留也不是分散
  5. 密码子在mRNA上，反密码子在tRNA上
  6. 生长素的两重性：低浓度促进，高浓度抑制
  7. 能量流动是单向的，不能循环利用
  8. 抗体由浆细胞产生，不是由T细胞产生
